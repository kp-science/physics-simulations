1. หลักการซ้อนทับ (Superposition Principle)

เมื่อคลื่นสองขบวน (หรือมากกว่า) ผ่านบริเวณเดียวกันในเวลาเดียวกัน การกระจัด (y) ของอนุภาคในตัวกลาง ณ จุดและเวลาใด ๆ มีค่าเท่ากับผลบวกเชิงพีชคณิต ของการกระจัดที่เกิดจากคลื่นแต่ละขบวนอยู่อย่างอิสระ

y(x, t) = y₁(x, t) + y₂(x, t)

ภาพจำลอง: คลื่นรวม (สีเหลือง) = ค่า y ของส้ม + ค่า y ของฟ้า ณ x เดียวกัน

2. กรณีที่พบบ่อย

🟠 เสริมเต็มที่ (Constructive)
ยอด + ยอด หรือ ท้อง + ท้อง · A_รวม = A₁ + A₂ (ขนาด)
เช่น A = +3 cm พบ +3 cm → +6 cm
🔴 หักล้าง (Destructive)
ยอด + ท้อง · ถ้า A₁ = |A₂| → A_รวม = 0 ชั่วขณะ
คลื่นไม่หาย — ผ่านกันแล้วคืนรูปเดิม
🟡 ขนาดต่างกัน
|A₁| ≠ |A₂| · ผลรวม = ผลบวกพีชคณิตเสมอ
เช่น +5 + 2 = +7 cm; +5 + (−2) = +3 cm

3. คุณสมบัติสำคัญ — คลื่นไม่ "ชน" กัน

หลังจากคลื่นดล 2 ลูกผ่านพ้นกัน แต่ละลูกยังคงรูปร่าง · ขนาด · ความเร็ว · ทิศทางเดิม ไม่มีการสูญเสียพลังงานหรือเปลี่ยนรูปจากการพบกัน นี่ต่างจากการชนกันของวัตถุ (อนุภาค)

ก่อนพบ → ขณะพบ (ซ้อนทับ) → หลังพ้น (รูปเดิม)

4. ความสัมพันธ์กับเฟส · y-t · y-x (เชื่อม Lab 39)

คลื่นดล (pulse) เป็นการรบกวนสั้น ๆ ที่ไม่ซ้ำเป็นคาบ จึงไม่ได้กำหนดเฟสแบบมุมในวงกลม แต่หลักการซ้อนทับยังใช้ได้เหมือนกันกับ คลื่นต่อเนื่องที่มีเฟส

คลื่นดล (Lab 40 นี้)
y(x,t) = f(x − vt) + g(x + vt)
ไม่มีคาบ, ไม่มีเฟส — บวกตามรูปร่างโดยตรง
คลื่นต่อเนื่อง (Lab 39)
y₁ = A sin(ωt − kx), y₂ = A sin(ωt − kx + Δφ)
A_รวม = 2A · |cos(Δφ/2)|
Δφ=0 → เสริม · Δφ=π → หักล้าง

วงกลมเฟสและการฉายเป็นกราฟ y-t · ใช้กับคลื่นต่อเนื่องเท่านั้น (สำหรับคลื่นดล ใช้รูปร่างโดยตรง)

5. ความเข้าใจคลาดเคลื่อน (Misconceptions)

M4.1 "หลังคลื่นพบกัน รูปจะเปลี่ยนถาวร / ผสมกัน" → ไม่จริง คลื่นแต่ละลูกคืนรูปเดิม 100%
M4.2 "ยอด+ท้อง = คลื่นหายไปจริง" → ไม่จริง · เป็นการหักล้าง ชั่วขณะ เท่านั้น พลังงานยังเก็บในความเร็วของอนุภาค
M4.3 "คลื่นใหญ่จะกลืนคลื่นเล็ก" → ไม่จริง · ทั้งสองผ่านกันได้โดยไม่รบกวน
M4.4 "คลื่นรูปต่างกันบวกกันไม่ได้" → ไม่จริง · บวกที่จุด x เดียวกันแบบพีชคณิตเสมอ
