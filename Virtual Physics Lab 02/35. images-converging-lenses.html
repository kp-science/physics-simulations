วัตถุประสงค์

ศึกษาภาพที่เกิดจากเลนส์นูน (converging lens / double convex lens) โดยแบ่งเป็น 2 ส่วน:

  • Part A: หาความยาวโฟกัส (focal length) ของเลนส์นูน
  • Part B: ศึกษาลักษณะภาพเมื่อวัตถุอยู่ที่ระยะต่างๆ จากเลนส์

อุปกรณ์

  • เลนส์นูน (double convex lens) เส้นผ่านศูนย์กลาง ~3 cm, ทางยาวโฟกัส ~10 cm
  • ม้านั่งเชิงแสง (optical bench) หรือ ไม้เมตร (meter stick) 100 cm
  • จอรับภาพ (screen) — กระดาษแข็งสีขาวหรือแผ่นพลาสติก
  • หลอดไฟเล็ก (light bulb) พร้อมขาตั้ง — ใช้เป็นวัตถุ
  • ตัวยึดเลนส์ (lens holder) และตัวยึดจอ (screen holder)

Part A — หาความยาวโฟกัสของเลนส์นูน

หลักการ

ความยาวโฟกัสของเลนส์ คือ ระยะจากเลนส์ถึงจุดที่รังสีขนานจากวัตถุไกลมากรวมกัน (focus) บนจอ

วิธีการทดลอง

  1. ลากเลนส์และจอวางบนม้านั่งเชิงแสง
  2. แสงอาทิตย์ (จำลองจากด้านซ้าย) ส่องผ่านเลนส์ไปยังจอ
  3. เลื่อนจอไปมา — ดูกล่อง "วงแสงบนจอ" มุมขวาบน
  4. เมื่อวงแสงเล็กที่สุด (เป็นจุด) = ตำแหน่งจุดโฟกัส
  5. ใช้ ไม้บรรทัด (↔ วัดระยะ) วัดระยะจากเลนส์ถึงจอ = f
  6. กรอกค่า f ที่วัดได้ลงตาราง แล้วกด "บันทึก"
  7. กด "สุ่มเลนส์ใหม่" เพื่อเปลี่ยนค่า f แล้วทำซ้ำอีก 2 ครั้ง (รวม 3 trials)
⚠️ ข้อควรระวัง: ถ้าใช้ดวงอาทิตย์จริง ระวังอย่าให้แสงโฟกัสเผาจอหรือมือ!

Part B — ศึกษาภาพที่ระยะวัตถุต่างๆ

หลักการ

เมื่อวัตถุอยู่ที่ระยะต่างๆ จากเลนส์นูน ภาพที่เกิดขึ้นจะมีลักษณะต่างกัน ทั้งตำแหน่ง ขนาด และทิศทาง สามารถตรวจสอบได้ด้วยสมการเลนส์บาง: 1/f = 1/do + 1/di

ตั้งค่าเลนส์

ใช้ slider ในแผงขวา ปรับ ระยะโฟกัส f และ ความสูงวัตถุ h ตามต้องการ

วิธีการทดลอง — ภาพจริง (do > f)

  1. ลากวัตถุ (แท่งสีส้ม) วางที่ระยะ do > f จากเลนส์
  2. ลากจอเลื่อนไปมา — ดูกล่อง "ภาพบนจอ" มุมขวาบน
  3. เมื่อภาพบนจอคมชัดที่สุด = จอตรงจุดเกิดภาพ
  4. ใช้ ไม้บรรทัด (↔ วัดระยะ) วัด do และ di
  5. ใช้ ไม้บรรทัด (↕ วัดความสูง) วัด ho และ hi
  6. กรอก 4 ค่าลงช่อง แล้วกด "บันทึก"

วิธีการทดลอง — ภาพเสมือน (do < f)

  1. ลากวัตถุเข้าใกล้เลนส์ให้ do < f
  2. จอจับภาพไม่ได้ — จะปรากฏ ดวงตา 👁 ด้านซ้ายเลนส์
  3. ลากดวงตาไปมาจนเห็นภาพชัดในกล่อง "มุมมองของตา"
  4. ใช้ไม้บรรทัดวัด do, di, ho, hi
  5. ⚠️ สำหรับภาพเสมือน: di และ hi ใส่เครื่องหมายลบ (−)
  6. กรอกค่าแล้วกด "บันทึก"
💡 เทคนิค: ลองวางวัตถุที่ระยะต่างๆ — do > 2f, = 2f, ระหว่าง f กับ 2f, = f, < f
⚠️ เมื่อ do ≈ f จะหาภาพยากมาก (ภาพอยู่ไกลมากหรือไม่เกิดเลย)

วิธีบันทึกผล (โหมดอัตโนมัติ)

  1. ใช้ slider ปรับ do (ระยะวัตถุ) และ f (ทางยาวโฟกัส)
  2. ระบบจะคำนวณ di, M, ชนิดภาพ อัตโนมัติ พร้อมแสดงรังสี 3 เส้น
  3. กด "บันทึก" เพื่อเพิ่มจุดลงกราฟ 1/do vs 1/di
  4. กด "CSV" เพื่อส่งออกข้อมูล

1. รังสีหลัก 3 เส้น (Principal Rays)

เลนส์นูน (converging lens) หักเหแสงให้รวมกัน เราใช้รังสีหลัก 3 เส้นจากปลายวัตถุเพื่อหาตำแหน่งภาพ:

  • รังสี 1 (แดง): ขนานแกนหลัก → ผ่านเลนส์ → หักเหผ่านจุดโฟกัสหลัง F'
  • รังสี 2 (เขียว): ผ่านจุดศูนย์กลางเลนส์ → ตรงไปไม่หักเห
  • รังสี 3 (น้ำเงิน): ผ่านจุดโฟกัสหน้า F → ผ่านเลนส์ → ออกขนานแกนหลัก

จุดที่รังสีทั้ง 3 ตัดกัน คือ ตำแหน่งภาพ

2. ภาพ 5 กรณี ตามระยะวัตถุ

ลักษณะของภาพเปลี่ยนตามระยะวัตถุ do เทียบกับ f:

กรณีระยะวัตถุ doระยะภาพ diชนิดทิศทางขนาด
1do > 2ff < di < 2fจริงกลับหัวเล็กกว่า
2do = 2fdi = 2fจริงกลับหัวเท่ากัน
3f < do < 2fdi > 2fจริงกลับหัวใหญ่กว่า
4do = f∞ไม่เกิดภาพ (รังสีขนาน)
5do < f|di| > doเสมือนหัวตั้งใหญ่กว่า

3. สมการเลนส์บาง (Thin Lens Equation)

1/f = 1/do + 1/di

โดยที่:

  • f = ทางยาวโฟกัส (positive สำหรับเลนส์นูน)
  • do = ระยะวัตถุ (positive เมื่อวัตถุอยู่ด้านหน้าเลนส์)
  • di = ระยะภาพ (positive = ภาพจริง, negative = ภาพเสมือน)
M = hi/ho = −di/do

กำลังขยาย (Magnification):

  • |M| > 1 → ภาพใหญ่กว่าวัตถุ
  • |M| = 1 → ภาพขนาดเท่าวัตถุ
  • |M| < 1 → ภาพเล็กกว่าวัตถุ
  • M < 0 → ภาพกลับหัว (inverted)
  • M > 0 → ภาพหัวตั้ง (erect)

กราฟ 1/do vs 1/di

จากสมการ: 1/di = −(1/do) + 1/f

เมื่อ plot 1/do บนแกน x กับ 1/di บนแกน y จะได้เส้นตรง:

  • ความชัน (slope) = −1
  • จุดตัดแกน y = 1/f
  • จุดตัดแกน x = 1/f

ดังนั้น จากกราฟสามารถหาค่า f ได้จากจุดตัดแกน
