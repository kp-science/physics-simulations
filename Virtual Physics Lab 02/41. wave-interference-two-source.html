🧪 การทดลองที่ 1 — วัดผลต่างทางเดิน (Path difference)

หลักการ: สำหรับแหล่งกำเนิดสองจุดที่สั่นเฟสตรงกัน จุดที่เป็นแนวปฏิบัพลำดับ n จะมี |S₁P − S₂P| = nλ และจุดที่เป็นแนวบัพลำดับ n จะมี |S₁P − S₂P| = (n − ½)λ

1
เลือกประเภทแหล่งกำเนิด (ช่องแคบคู่ / จุดสั่น 2 อัน) · ถ้าเป็นจุดสั่น ให้เลือกเฟส (ตรงกัน/ตรงข้าม)
2
ตั้งค่า d และ f · กด 🎲 สุ่มความเร็ว เพื่อให้โปรแกรมซ่อน v จริง (โหมดทดลองเสมือน) หรือกำหนดค่า v เองเพื่อเช็คผล
3
กด ▶ เริ่ม · สังเกตรูปแบบการแทรกสอด บริเวณสีสว่าง/มืดสลับ
4
เลือกว่าจะวัด ปฏิบัพ (A) หรือ บัพ (N) · แล้วคลิกจุด Pบน canvas ที่คิดว่าอยู่บนแนวนั้น
5
โปรแกรมจะวาดเส้น S₁P, S₂P และแสดงค่าระยะ · ระบุ ลำดับ n ของแนวที่เลือก (0 = กลาง, 1 = ถัดไป…)
6
กด ➕ บันทึกจุดนี้ โปรแกรมจะคำนวณ λ จากสูตร แล้วบันทึกลงตาราง
7
ทำซ้ำ ≥ 4–6 จุด คนละแนว (ทั้งบัพและปฏิบัพ หลาย ๆ ลำดับ) · โปรแกรมสรุปค่าเฉลี่ย λ และ %error เทียบค่าจริง
💡 โหมดบันทึกเอง: ฝึกการอ่านและตัดสินใจว่าจุดที่คลิกอยู่บนแนวใดกันแน่ · ถ้าอยากดูเฉลย ให้ติ๊ก "แสดง hint แนวบัพ/ปฏิบัพ" แล้วจะเห็นเส้นโค้ง hyperbola ทับเส้นจริง
⚡ โหมดบันทึกอัตโนมัติ: กดปุ่ม "เติมค่าครบ" โปรแกรมสร้างจุด A₀, A₁, A₂, N₁, N₂ อย่างถูกต้องให้ทันที — ใช้ตรวจสอบว่าทำด้วยมือได้ผลตรงกับทฤษฎีหรือไม่

🧪 การทดลองที่ 2 — วัดมุม (Angular method)

หลักการ (ฟาร์ฟิลด์): ที่ระยะไกลจากแหล่งกำเนิด แนวปฏิบัพลำดับ n ทำมุม θ กับแนวกลางโดย d sinθ = nλ (เฟสตรงกัน) · แนวบัพลำดับ n ทำมุม d sinθ = (n − ½)λ

1
เปิดไม้โปรแทกเตอร์ (ติ๊ก "แสดงไม้โปรแทกเตอร์") · ศูนย์กลางของโปรแทกเตอร์อยู่ที่ จุด M กลางระหว่างแหล่งกำเนิด 2 อัน แนว 0° = แนวกลาง
2
เลือกประเภท (บัพ/ปฏิบัพ) และคลิกจุด P บนแนวที่ต้องการวัด (ควรห่างจาก M ไกล ๆ เพื่อให้ประมาณฟาร์ฟิลด์แม่นขึ้น)
3
อ่านมุม θ จากไม้โปรแทกเตอร์ (เส้นสีม่วงจาก M ไป P) แล้วกรอกลงช่อง "มุม θ อ่านจากไม้โปรแทกเตอร์"
4
ระบุ n · กด ➕ บันทึกจุดนี้ โปรแกรมคำนวณ λ จากสูตรที่เหมาะสมและเทียบกับค่าจริง
5
ทำซ้ำหลายมุม · สังเกตว่ายิ่ง n มากมุมก็ยิ่งกว้าง · สรุป λ เฉลี่ย
📏 วิธีอ่านมุม: วางสายตาให้ตรงกับเส้นสีม่วง (M→P) แล้วอ่านตัวเลขบนไม้โปรแทกเตอร์ที่เส้นตัด · ค่าตัวเลขใต้ตำแหน่ง "มุม θ (โปรแกรมวัด)" คือเฉลยสำหรับตรวจงาน
⚠️ ข้อควรระวัง: สูตร d sinθ = nλ เป็นการประมาณฟาร์ฟิลด์ (r ≫ d) · ถ้าคลิกจุด P ใกล้แหล่งกำเนิดมากจะมี %error สูงกว่าการทดลอง 1

🧪 การทดลองที่ 3 — ช่องแคบเดียว (Single-slit diffraction)

หลักการ: คลื่นผ่านช่องกว้าง a · ตามหลักฮอยเกนส์ ทุกจุดในช่องเป็นแหล่งกำเนิด wavelet · เกิดการแทรกสอดภายใน ทำให้เกิด central maximum (แถบสว่างกว้างตรงกลาง) และ minima (แถบมืด n=1,2,3…) สลับกับ secondary maxima

แหล่งกำเนิดเทียบวัด (S₁, S₂) อยู่ที่จุดกึ่งกลางของแต่ละครึ่งช่อง (y = ±a/4) ไม่ใช่ที่ขอบ · ระยะระหว่าง S₁ และ S₂ = a/2

วิธีวัดเหมือน 2 การทดลองก่อน เพียงเปลี่ยนสูตร:

Path diff:  |Δr| = (n/2)λ (แถบมืด/บัพ)  ·  |Δr| = ((n+½)/2)λ (secondary max/ปฏิบัพ)
Angular:   (a/2) sinθ = (n/2)λ  ⇔  a sinθ = nλ (แถบมืด)  ·  a sinθ ≈ (n+½)λ (secondary)
🔍 ทำไมใช้จุดกึ่งกลางแต่ละครึ่ง? การจับคู่ wavelet ของ Fraunhofer: wavelet ทุกตัวในครึ่งล่างจับคู่กับ wavelet ที่อยู่ห่างออกไป a/2 ในครึ่งบน · "ตัวแทน" ของครึ่งแต่ละฝั่งคือจุดกึ่งกลาง (y = ±a/4) · ใช้จุดนี้เป็นหลักในการวัด r₁, r₂ จะให้ผลเดียวกับสูตร a sinθ = nλ
1
เลือกประเภทแหล่งกำเนิด "ช่องแคบเดียว" · slider "กว้างช่อง a" จะปรากฏแทน "ระยะ d"
2
ตั้งค่า a (ควรใหญ่กว่า λ อย่างน้อย 2 เท่าเพื่อให้เห็น minima อย่างน้อย 1 คู่) และ f · เริ่มการจำลอง
3
สังเกต central maximum (สว่างและกว้างตรงกลาง) และบริเวณแถบมืดข้างเคียง · (ติ๊ก hint เพื่อเปรียบเทียบ)
4
คลิกจุด P บนแถบมืดลำดับ n=1 · เลือก "บัพ (N)" · ระบุ n=1 · บันทึก → λ = |Δr| / 1 (Path diff) หรือ a sinθ / 1 (Angular)
5
ทำซ้ำกับ secondary maxima: เลือก "ปฏิบัพ (A)" ระบุ n=1, 2 · λ = |Δr| / (n+½)
6
ทดลองลด a ให้ใกล้ λ · สังเกตว่า minima หายไป (a ≤ λ) → คลื่นกระจายเหมือนแหล่งกำเนิดจุด
💡 เคล็ดลับ: ขอบของ central max คือ minimum แรก · ระยะจาก center-line ไปยังแนวมืดแรกคือ "ครึ่งความกว้าง" ของ central max (sinθ = λ/a)
⚠️ หมายเหตุ: สูตร (n+½)λ สำหรับ secondary max เป็นการประมาณ · ค่าที่แม่นจริงได้จาก tan(φ)=φ (φ = πa sinθ/λ) ซึ่งให้ φ₁≈4.493, φ₂≈7.725 → a sinθ ≈ 1.430λ, 2.459λ ตามลำดับ (ไม่ใช่ 1.5λ, 2.5λ เป๊ะ) · ในการทดลองระดับมัธยม ใช้ (n+½) เพียงพอ

📊 การวิเคราะห์ผลและสรุป

  • λ ที่วัดได้ (เฉลี่ย) ควรใกล้เคียงค่าจริง v/f · หาก %error > 15% ให้วิเคราะห์ว่ามาจากการอ่านค่าผิด n หรือคลิกจุดเพี้ยน
  • เปลี่ยน d · สังเกตว่าระยะระหว่างแนวบัพ/ปฏิบัพบน canvas แคบลงเมื่อ d เพิ่ม
  • เปลี่ยน f · สังเกตว่ารูปแบบเปลี่ยนไปอย่างไร (λ สั้นลง ⇒ แนวถี่ขึ้น)
  • เปรียบเทียบช่องแคบคู่ (เฟสตรงกันเสมอ) กับจุดสั่น 2 อัน เฟสตรงข้าม · แนวกลาง (θ=0) จาก "ปฏิบัพ A₀" กลายเป็น "บัพ N₀"

📝 คำถามท้ายการทดลอง

  1. ถ้าเพิ่ม d เป็น 2 เท่าโดยให้ f คงที่ มุม θ ของแนวปฏิบัพลำดับ n=1 จะเปลี่ยนอย่างไร?
  2. เพราะเหตุใดการแทรกสอดจากช่องแคบคู่ จึงเหมือนกับแหล่งกำเนิดจุด 2 อันที่มีเฟสตรงกัน?
  3. หากจุดสั่น 2 อันมีเฟสตรงข้าม เหตุใดแนวกลาง (θ=0) จึงกลายเป็นแนวบัพ?
  4. ระหว่างวิธีวัด Path Difference และวัดมุม วิธีใดน่าจะให้ค่า λ แม่นกว่ากัน และเพราะอะไร?

1. หลักการซ้อนทับและการแทรกสอด

เมื่อมีคลื่นสองขบวนเดินทางผ่านจุดเดียวกัน การกระจัดรวมคือผลบวกทางพีชคณิตของการกระจัดจากแต่ละขบวน (หลักการซ้อนทับ / Superposition):

y(x,y,t) = y₁(x,y,t) + y₂(x,y,t)

เมื่อสองขบวนเสริมกัน (ยอด + ยอด) → แอมพลิจูดรวมสูงสุด เรียกว่า การแทรกสอดแบบเสริม (Constructive) · เมื่อสวนทาง (ยอด + ท้อง) → แอมพลิจูดรวมน้อยสุด เรียกว่า การแทรกสอดแบบหักล้าง (Destructive)

2. แหล่งกำเนิดอาพันธ์ (Coherent sources)

แหล่งกำเนิดสองอันจะให้รูปแบบการแทรกสอดที่คงที่ได้ เมื่อมีความถี่เดียวกัน และผลต่างเฟสคงที่ เรียกว่า แหล่งกำเนิดอาพันธ์

  • ช่องแคบคู่ (Double slit): คลื่นระนาบเดียวกันวิ่งชนตัวกั้น ลอดผ่าน 2 ช่องพร้อมกัน → ออกมาเฟสตรงกันเสมอ (หลักของฮอยเกนส์)
  • จุดสั่น 2 อัน: ใช้มอเตอร์/ตัวสั่นคู่ ปรับเฟสได้ทั้ง "ตรงกัน" (เคลื่อนลง-ขึ้นพร้อมกัน) หรือ "ตรงข้าม" (สวนกัน)

3. เงื่อนไขการแทรกสอด

ให้ S₁, S₂ อยู่ห่างกัน d · P เป็นจุดในสนามคลื่น โดย r₁ = S₁P, r₂ = S₂P · ผลต่างทางเดิน Δr = |r₁ − r₂|

🟢 กรณีเฟสตรงกัน (In-phase)
ปฏิบัพ (Constructive, antinodal): Δr = nλ    n = 0, 1, 2, 3…
บัพ (Destructive, nodal): Δr = (n − ½)λ    n = 1, 2, 3…
แนวกลาง (Δr=0) = ปฏิบัพลำดับ 0 (A₀)
🟠 กรณีเฟสตรงข้าม (Anti-phase)
ปฏิบัพ: Δr = (n − ½)λ    n = 1, 2, 3…
บัพ: Δr = nλ    n = 0, 1, 2, 3…
แนวกลาง กลายเป็นบัพลำดับ 0 (N₀)

ภาพ: แนวปฏิบัพ (เขียว) และแนวบัพ (ฟ้า) เป็นเส้นโค้ง hyperbola ที่มีโฟกัสที่ S₁, S₂

4. สูตรฟาร์ฟิลด์ (มุม θ)

เมื่อระยะไปจุด P ไกลกว่า d มาก (r ≫ d) เส้น S₁P และ S₂P แทบจะขนานกัน ผลต่างทางเดินประมาณเป็น d sinθ เมื่อ θ เป็นมุมจากแนวกลาง:

d sinθ = nλ (ปฏิบัพ, in-phase)
d sinθ = (n − ½)λ (บัพ, in-phase)

ภาพ: เส้นจาก S₁, S₂ เกือบขนานเมื่อไกล · ผลต่างทางเดินคือโครงสามเหลี่ยมมุมฉากขนาดเล็ก d sinθ

5. ผลของตัวแปร

เพิ่ม d (ระยะระหว่างแหล่งกำเนิด)
มุม θ ของแนวลำดับเดิมแคบลง (sinθ = nλ/d ⇒ θ ↓) · แนวถี่ขึ้น
เพิ่ม f (ความถี่)
λ = v/f ลดลง · แนวถี่ขึ้นเช่นกัน · มีลำดับ n สูงสุด n_max = ⌊d/λ⌋ (ปฏิบัพ)

6. ความสัมพันธ์กับ Young's Double-Slit (แสง)

สมการที่ใช้เหมือนกัน เพียงเปลี่ยนคลื่นผิวน้ำ → คลื่นแสง · ระยะแถบบนจอ y ≈ L·tanθ ≈ L·sinθ ⇒ y = nλL/d

7. ช่องแคบเดียว — การเลี้ยวเบน (Single-slit Diffraction)

เมื่อคลื่นระนาบลอดผ่านช่องแคบช่องเดียวที่กว้าง a · ตามหลักฮอยเกนส์ ทุกจุดบนหน้าคลื่นในช่องทำหน้าที่เป็นแหล่งกำเนิดย่อย (wavelet) · คลื่นที่เดินทางออกไปในทิศต่าง ๆ จะแทรกสอดกันเอง เกิดรูปแบบ central maximum กว้าง + secondary maxima และ minima (ช่องมืด) สลับ

การจับคู่ wavelet (Pairing method): แบ่งช่องเป็นสองครึ่ง — ครึ่งบน (0 ถึง +a/2) และครึ่งล่าง (−a/2 ถึง 0) · wavelet ที่ตำแหน่ง y ในครึ่งล่าง จับคู่กับ wavelet ที่ y + a/2 ในครึ่งบน · ระยะห่างระหว่างคู่ = a/2

ในการจำลองนี้ เราใช้จุดกึ่งกลางของแต่ละครึ่งเป็นแหล่งกำเนิดเทียบวัด (effective source) S₁ = (x, −a/4), S₂ = (x, +a/4) · ระยะระหว่าง S₁, S₂ จึงเท่ากับ a/2 · ผลต่างทางเดินที่มุม θ (ฟาร์ฟิลด์):

Δr_วัด = (a/2) sinθ     (ระยะระหว่างครึ่งช่อง × sinθ)

เมื่อคู่ wavelet ทุกคู่ภายในช่องหักล้างกันพอดี (ผลต่างเฟส = π) จะเกิด minimum:

Δr_วัด = nλ/2  ⇔  (a/2)sinθ = nλ/2  ⇔  a sinθ = nλ

และเงื่อนไขสรุป:

🔵 Minima (แถบมืด / บัพ)
a sinθ = nλ    n = 1, 2, 3, …
คู่ wavelet ภายในช่องจับคู่หักล้างกันครบทั้งช่อง → ความเข้มเฉลี่ย = 0
🟢 Maxima (แถบสว่าง / ปฏิบัพ)
Central max: θ = 0 (สว่างสุด, กว้างสุด)
Secondary: a sinθ ≈ (n + ½)λ    n = 1, 2, 3, …
*ค่าจริงของ secondary max ได้จากสมการ tan(φ) = φ (φ = πa sinθ/λ) สูตร (n+½) เป็นค่าประมาณที่ใช้ได้ดีสำหรับ n ≥ 2

ความแตกต่างสำคัญจากช่องแคบคู่:

  • ช่องแคบคู่ (Young's): d sinθ = nλ → ปฏิบัพที่ n คือจำนวนเต็ม · Central (n=0) = ปฏิบัพ
  • ช่องแคบเดียว (diffraction): a sinθ = nλ → บัพ (dark) ที่ n = 1, 2, 3... · Central (n=0) = ปฏิบัพสว่างสุด (กว้าง)
  • สังเกต: เครื่องหมายสลับกัน! ค่า n ที่เป็นเงื่อนไขปฏิบัพในช่องแคบคู่ กลายเป็นเงื่อนไขบัพในช่องแคบเดียว

ภาพ: การจับคู่ wavelet · แต่ละ wavelet ในครึ่งล่างจับคู่กับ wavelet ที่อยู่ห่างออกไป a/2 ในครึ่งบน · ผลต่างทางเดินของคู่ = (a/2) sinθ · เมื่อ = λ/2 คู่หักล้างกัน (dark minimum แรก ⇒ a sinθ = λ)

ความกว้างของ central maximum

ขอบของ central max คือ minimum แรก (n=1) ทั้งสองข้าง → ช่วงมุม sinθ = ±λ/a · ยิ่ง a เล็ก central max ยิ่งกว้าง (เลี้ยวเบนมากขึ้น) · ถ้า a ≤ λ จะไม่เกิด minimum เลย คลื่นกระจายเหมือนแหล่งกำเนิดจุดเดียว

ภาพ: กราฟความเข้ม I(θ) = I₀ sinc²(πa sinθ/λ) · Central max กว้างสุด (±λ/a) · secondary maxima เล็กกว่าและแคบกว่า · minima ที่ a sinθ = nλ (n = 1, 2, 3,…)

8. Misconception ที่พบบ่อย

  • ❌ แนวบัพ = คลื่นหายไป · ✅ ความจริงคือคลื่นสองขบวนยังมาถึง แต่หักล้างกัน (พลังงานเฉลี่ย = 0)
  • ❌ ปฏิบัพลำดับ 0 อยู่ข้างเดียว · ✅ A₀ คือแนวกลาง (แกนสมมาตร)
  • ❌ d มากขึ้น แนวกว้างขึ้น · ✅ ตรงข้าม — d มากขึ้น แนวถี่/แคบลง
  • ❌ เฟสตรงข้ามจะไม่เกิดการแทรกสอด · ✅ เกิดได้ปกติ เพียงแต่แนวบัพและปฏิบัพสลับที่กัน
