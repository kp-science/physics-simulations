📦 อุปกรณ์ (ใน Simulation)

  • ตัวส่งคลื่นไมโครเวฟ (TX) — แสดงเป็นวงกลม ไม่มีทิศทาง เพื่อไม่ให้นักเรียนทราบทิศรังสี (λ ≈ 3 cm)
  • แผ่นสะท้อน (Reflector) — แผ่นโลหะ 20×20 cm
  • ตัวรับสัญญาณ (RX) — แสดงเป็นสี่เหลี่ยม ลากได้ 2 แกน (ราบ+ดิ่ง) พร้อมมิเตอร์วัดสัญญาณ

🔬 วิธีการทดลอง — โหมดบันทึกเอง

  1. กด "🎲 สุ่มใหม่" — ระบบสุ่มมุมตกกระทบ (θ_H = 15°–75°) และมุมระนาบ (θ_V = ±18°)
  2. สังเกต — ตัวส่ง (วงกลม) ปรากฏในมุมมองทั้ง 2 แต่ ไม่แสดงทิศของรังสี
  3. ลาก RX ในมุมมองด้านบน (ซ้าย-ขวา) → ปรับมุมแนวราบ
  4. ลาก RX ในมุมมองด้านข้าง (บน-ล่าง) → ปรับมุมแนวดิ่ง
  5. ดูมิเตอร์สัญญาณ → ค่อยๆ หาจุดที่สัญญาณแรงสุด (peak ≈ 100%)
  6. อ่านค่า φ — panel แสดง φ แนวราบ และ φ แนวดิ่ง (มุมระหว่าง TX กับ RX วัดจาก reflector)
  7. คำนวณ θ = φ / 2 — panel คำนวณให้อัตโนมัติ
  8. กด "✓ เสร็จสิ้น" → ล็อกตำแหน่ง → กด "📝 บันทึก" → ระบบเปิดเผยค่าจริง + %Error
  9. ทำซ้ำ 5–8 ครั้ง ตรวจสอบว่า θ ที่วัดได้ ≈ ค่าจริง

⚡ โหมดอัตโนมัติ

  1. สลับไปโหมด "⚡ อัตโนมัติ" → ตั้งจำนวนครั้ง → กด "▶ สแกน"
  2. ระบบสุ่มมุม สแกนหา peak อัตโนมัติ (มี noise เล็กน้อย) แล้วบันทึกทุกครั้ง

📡 คลื่นไมโครเวฟ

คลื่นไมโครเวฟเป็นคลื่นแม่เหล็กไฟฟ้า ความยาวคลื่น 1 mm – 30 cm (1–300 GHz) ในการทดลองนี้ใช้ λ ≈ 3 cm (~10 GHz) มีสมบัติเหมือนแสง — สะท้อน หักเห เลี้ยวเบน — แต่ λ ใหญ่กว่ามาก จึงสังเกตได้ง่าย

🪞 กฎการสะท้อน

  1. รังสีตกกระทบ Normal และรังสีสะท้อนอยู่ในระนาบเดียวกัน
  2. มุมตกกระทบ = มุมสะท้อน (θᵢ = θᵣ)
θᵢ = θᵣ

🔄 ความสัมพันธ์ φ กับ θ

φ = มุมรวมระหว่างรังสีตกกระทบกับรังสีสะท้อน (วัดจากตำแหน่ง TX ถึง RX ผ่าน reflector)

φ = θᵢ + θᵣ = 2θ  →  θ = φ / 2

สมบัตินี้ใช้ได้ทั้งแนวราบและแนวดิ่ง — ถ้าระนาบตกกระทบเอียง φ_V ≠ 0° กฎการสะท้อนยังใช้ได้เหมือนเดิมในทั้ง 2 มิติ

📈 ผลที่คาดหวัง

กราฟ θ_จริง vs θ_วัด เป็นเส้นตรง slope = 1 ผ่านจุดกำเนิด

🔢 %Error

%Error = |θ_จริง − θ_วัด| / θ_จริง × 100

ตัวอย่าง: θ_H จริง = 40°, วัดได้ 38° → %Error = 5.0%

เกณฑ์: %Error_H < 5% · |ΔV| < 3°
