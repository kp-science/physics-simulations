🧪 อุปกรณ์

1. เลเซอร์ (สุ่มความยาวคลื่น)
2. จอรับภาพ (ปรับระยะ L ได้)
3. ใบมีดโกน / สลิตเดี่ยว / สลิตคู่
4. เกรตติงเลี้ยวเบน (เลือกจำนวนเส้นได้)
5. หลอดแก๊ส He, Na (สำหรับเกรตติง)
6. เครื่องวัดตำแหน่ง (Probe)

A — การเลี้ยวเบนที่ขอบมีด (Knife-Edge)

  1. เลื่อน slider ให้ใบมีดบังลำแสงเลเซอร์ → สังเกต fringes บนจอ
  2. ลากเม้าส์บนจอเพื่อวัด intensity ที่ตำแหน่งต่างๆ
  3. กรอก position (mm) และ I/I₀ → กด บันทึก (สลับ สว่าง↔มืด อัตโนมัติ)
  4. กด 📈 ประมวลผล เพื่อเทียบกราฟที่วัดกับทฤษฎี Fresnel + คำนวณ %Error

B — สลิตเดี่ยว (Single Slit)

  1. ระบบสุ่ม λ ให้ (สีลำแสงเป็นคำใบ้) ปรับความกว้างช่อง a และระยะจอ L
  2. สังเกต pattern: แถบกลางสว่างสุดกว้างสุด แถบข้างเคียงจางลง
  3. ลากเม้าส์หาตำแหน่งแถบมืดลำดับที่ 1 (intensity ≈ 0 แรกจากกลาง)
  4. กรอก y₁ (cm) → กด บันทึก → ระบบคำนวณ λ = y₁·a/L เทียบกับค่าจริง
สูตรตำแหน่งแถบมืด (minima):
a·sin θ = mλ → ตำแหน่งบนจอ: y_m = mλL/a   (m = ±1, ±2, ...)
คำนวณ λ:
λ = y₁·a / L

C — สลิตคู่ (Double Slit)

  1. ระบบสุ่ม λ ให้ ปรับระยะห่างช่อง d, ความกว้างช่อง a, ระยะจอ L
  2. สังเกต pattern: แถบสว่าง-มืดสลับถี่ (interference) ภายใต้ diffraction envelope
  3. ลากเม้าส์หาตำแหน่งแถบสว่างลำดับที่ n จากแนวกลาง
  4. กรอก n และ y (cm) → กด บันทึก → ระบบคำนวณ λ = y·d/(n·L)
สูตรแถบสว่าง (constructive):
d·sin θ = nλ → ตำแหน่งบนจอ: y_n = nλL/d   (n = 0, ±1, ±2, ...)
คำนวณ λ:
λ = y·d / (n·L)

D — เกรตติง (Diffraction Grating)

โหมดเลเซอร์

  1. ระบบสุ่ม λ ให้ เลือกเกรตติง (จำนวนเส้น/cm) ปรับระยะจอ L
  2. สังเกต: แถบสว่างคมชัด (sharp maxima) ห่างเท่ากัน ทั้งสองด้าน
  3. ลากเม้าส์วัดระยะ y ถึงแถบสว่างลำดับ m → กด บันทึก
  4. ระบบคำนวณ λ = y·d/(m·L) เทียบกับค่าจริง

โหมดฮีเลียม / โซเดียม

  1. เลือกแหล่งกำเนิดเป็น He หรือ Na → เห็น emission lines หลายสีพร้อมกัน
  2. กดเลือกสีที่กำลังวัด → ลากเม้าส์วัด y → กด บันทึก
  3. ระบบคำนวณ λ เทียบกับ λ จริงของ emission line นั้น

โหมดแสงขาว

  • สังเกตสเปกตรัมสีรุ้งที่แต่ละ order (ม่วงใกล้กลาง แดงไกลกลาง)
  • m=0 เป็นสีขาว เพราะทุก λ มี constructive interference ที่ θ=0
สมการเกรตติง:
d·sin θ = mλ → y = mλL/d
d = 1/(จำนวนเส้นต่อ cm)   เช่น 5,276 gr/cm → d = 1.896 μm
จำนวน order สูงสุด:
n_max = floor(d/λ)   จำนวนแถบสว่างทั้งหมด = 2n_max + 1
คำนวณ λ:
λ = y·d / (m·L)    % Error = |λ_calc − λ_actual| / λ_actual × 100
⚠️ ข้อควรระวัง: ห้ามมองตรงเข้าไปในลำแสงเลเซอร์โดยเด็ดขาด!

1. หลักการของฮอยเกนส์ (Huygens' Principle)

ทุกจุดบนหน้าคลื่นเป็นแหล่งกำเนิดคลื่นทุติยภูมิ เมื่อแสงผ่านช่องเปิดแคบ แต่ละจุดจะแผ่คลื่นออกทุกทิศ → เกิดปรากฏการณ์การเลี้ยวเบน (Diffraction)

2. การเลี้ยวเบนที่ขอบมีด (Exp A)

ขอบเงาไม่คมชัด แต่มี fringes สว่าง-มืดสลับ ที่ขอบเงาพอดี I = I₀/4

Fresnel Diffraction:
I(y) = (I₀/2) × [(C(v)+½)² + (S(v)+½)²]
v = y√(2/λL)   |   y = ระยะจากขอบเงา   |   L = ระยะใบมีด→จอ

3. สลิตเดี่ยว — Fraunhofer Diffraction (Exp B)

แถบกลาง (central max) สว่างสุดกว้างสุด ตำแหน่งแถบมืดลำดับ m อยู่ที่ a·sinθ = mλ

I(θ) = I₀[sin(β)/β]²   |   β = πa·sinθ/λ

ตำแหน่งแถบมืด + คำนวณ λ:
y_m = mλL/a → λ = y₁·a/L

ช่องแคบลง → pattern กว้างขึ้น | จอไกลขึ้น → pattern กว้างขึ้น

4. สลิตคู่ — Double-Slit Interference (Exp C)

แสงจาก 2 ช่อง (ห่างกัน d) แทรกสอดกัน เกิดแถบสว่าง-มืดสลับถี่ ภายใต้ diffraction envelope ของช่องเดี่ยว

แถบสว่าง (Constructive):
d·sinθ = nλ → y_n = nλL/d   (n = 0, ±1, ±2, ...)

คำนวณ λ:
λ = y·d / (n·L)

Intensity = cos²(πd·sinθ/λ) × [sin(πa·sinθ/λ)/(πa·sinθ/λ)]²

5. เกรตติงเลี้ยวเบน (Exp D)

เกรตติงมีช่องจำนวนมาก (N ช่อง) ห่างกัน d เท่ากัน → แถบสว่างคมชัดมาก (sharp peaks) ห่างเท่ากัน

สมการเกรตติง (เหมือนสลิตคู่):
d·sinθ = mλ → y = mλL/d
d = 1/(จำนวนเส้นต่อ cm)   เช่น 5,276 gr/cm → d = 1.896 μm

จำนวน order สูงสุด:
n_max = ⌊d/λ⌋   |   จำนวนแถบสว่าง = 2n_max + 1

คำนวณ λ:
λ = y·d / (m·L)   |   %Error = |λ_calc − λ_actual|/λ_actual × 100

แสงขาว vs เลเซอร์ ผ่านเกรตติง

เลเซอร์ (monochromatic): แถบสว่างสีเดียวห่างเท่ากันทุก order
แสงขาว (polychromatic): แต่ละ order เป็นสเปกตรัมสีรุ้ง ม่วง (λ สั้น) ใกล้กลาง แดง (λ ยาว) ไกลกลาง m=0 เป็นสีขาว

Emission Spectra ผ่านเกรตติง

He: เห็น 6 เส้นสี (447, 471, 502, 588, 668, 707 nm) แยกกันที่แต่ละ order
Na: เห็น sodium doublet (D₁ = 589 nm, D₂ = 590 nm) — เส้นเหลืองคู่ชิดกัน
