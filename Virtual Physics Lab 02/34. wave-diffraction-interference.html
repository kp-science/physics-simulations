🔬 การทดลอง A — การเลี้ยวเบนที่ขอบมีด (Knife-Edge Diffraction)

📝 วิธีทำ

  1. ตั้งเลเซอร์ He-Ne ฉายไปยังจอที่ห่างประมาณ 1 เมตร
  2. ค่อยๆ เลื่อนใบมีดโกนเข้ามาบังลำแสงเลเซอร์จากด้านข้าง
  3. สังเกตเงาบนจอ — จะเห็นว่าขอบเงาไม่คมชัด แต่มีแถบสว่าง-มืดสลับกัน (fringes)
  4. แถบเหล่านี้เกิดจากการแทรกสอดของคลื่นแสงที่เลี้ยวเบนรอบขอบใบมีด
  5. ใน simulation: ใช้ slider "ตำแหน่งใบมีด" เลื่อนใบมีดเข้า-ออก แล้วสังเกต pattern บนจอ

📊 การบันทึกผล

  • สังเกตและบันทึกลักษณะ pattern ที่เห็นบนจอ (มีกี่แถบ, สว่าง-มืดอย่างไร)
  • ใน simulation: เลื่อน photometer บนจอเพื่ออ่าน intensity ที่ตำแหน่งต่างๆ

🔬 การทดลอง B — การเลี้ยวเบนผ่านช่องเดี่ยว (Single-Slit Diffraction)

📝 วิธีทำ

  1. ใช้ใบมีดโกน 2 ใบวางขนานกันสร้างเป็นช่องแคบ (slit)
  2. ฉายเลเซอร์ผ่านช่อง ไปยังจอห่างประมาณ 1 เมตร
  3. เลื่อนใบมีดเข้าใกล้กัน (ช่องแคบลง) → สังเกตว่าลำแสงบนจอกว้างขึ้น
  4. เมื่อช่องแคบมากๆ จะเห็น pattern เป็นแถบสว่าง-มืดสลับ โดยแถบกลางสว่างสุด
  5. ใช้โฟโตมิเตอร์วัดความเข้ม (intensity) ที่ตำแหน่งต่างๆ ห่างกันทีละ 1 cm ข้ามจากขอบหนึ่งไปอีกขอบ
  6. วาดกราฟ Intensity vs Distance

📊 การบันทึกผล

  • โหมดบันทึกเอง: เลื่อน photometer (เม้าส์บนจอใน canvas) อ่านค่าตำแหน่ง (cm) และ intensity แล้วกรอกลงตาราง
  • โหมดอัตโนมัติ: ระบบสแกน intensity ทั้ง pattern ให้อัตโนมัติ แสดงตาราง + กราฟ

🔬 การทดลอง C — เกรตติงเลี้ยวเบน (Diffraction Grating)

📝 วิธีทำ

  1. ตั้งจอห่าง 1 เมตร จากเลเซอร์ ทำเครื่องหมายจุดที่แสงตก
  2. วาง diffraction grating หน้าเลเซอร์
  3. สังเกตจุดสว่างเพิ่มเติมบนจอ: central maximum (m=0), 1st order (m=±1), 2nd order (m=±2), ...
  4. วัดระยะ x = ระยะจาก central maximum ถึง first order maximum (ทั้งซ้ายและขวา แล้วเฉลี่ย)
  5. วัดระยะ L = ระยะจากเกรตติงถึง first order maximum บนจอ
  6. คำนวณ λ = d × x / L
  7. เทียบกับค่าอ้างอิง λ = 632.8 nm → คำนวณ % Error
  8. ทำซ้ำหลายครั้งด้วยระยะ L ต่างกัน

📊 สูตรที่ใช้

λ = d × x / L
d = ระยะห่างระหว่าง slits บนเกรตติง (เช่น d = 1/5276 cm = 1.896 × 10⁻⁴ cm = 1.896 μm)
x = ระยะจาก central maximum ถึง first order maximum (m)
L = ระยะจากเกรตติงถึง first order maximum บนจอ (m)
ค่าอ้างอิง: λ(He-Ne) = 6.328 × 10⁻⁷ m = 632.8 nm
% Error = |λ_measured − λ_actual| / λ_actual × 100

⚠️ ข้อควรระวัง: ระวังอันตรายจากเลเซอร์ — ห้ามมองตรงเข้าไปในลำแสงเลเซอร์โดยเด็ดขาด! ใช้ใบมีดโกนใหม่เพื่อให้ขอบคมและสร้าง pattern ที่ชัดเจน

1. หลักการของฮอยเกนส์ (Huygens' Principle)

ทุกจุดบนหน้าคลื่น (wavefront) สามารถถือเป็นแหล่งกำเนิดคลื่นทุติยภูมิ (secondary source) ที่แผ่คลื่นทรงกลมออกไป หน้าคลื่นใหม่เกิดจากผลรวม (envelope) ของคลื่นทุติยภูมิเหล่านี้

เมื่อแสงผ่านช่องเปิดแคบ จุดทุกจุดบนช่องเปิดจะกลายเป็นแหล่งกำเนิดคลื่นทุติยภูมิ ทำให้แสงแผ่ออกทุกทิศทาง — นี่คือปรากฏการณ์การเลี้ยวเบน (Diffraction)

2. การเลี้ยวเบนที่ขอบมีด (Knife-Edge / Fresnel Diffraction)

เมื่อแสงเลเซอร์กระทบขอบมีดโกน ขอบเงาบนจอจะไม่คมชัด แต่จะมีแถบสว่าง-มืดสลับกัน (fringes) เรียกว่า Fresnel diffraction fringes

ที่ขอบเงาพอดี ความเข้มแสง = I₀/4 (หนึ่งในสี่ของความเข้มเดิม) ฝั่งที่ไม่ถูกบัง (illuminated side) จะมี fringes ที่สว่างกว่า I₀ แล้วค่อยๆ ลดลงมาที่ I₀ ส่วนฝั่งเงา (shadow side) ความเข้มลดลงอย่างรวดเร็วเข้าสู่ศูนย์

Fresnel Diffraction ที่ขอบมีด:
ใช้ Fresnel Integrals C(u), S(u) คำนวณ:
I(y) = (I₀/2) × [(C(v) + ½)² + (S(v) + ½)²]

v = y × √(2 / λL)
y = ระยะจากขอบเงา (+ = ฝั่งสว่าง, − = ฝั่งมืด)
L = ระยะจากใบมีดถึงจอ
λ = 632.8 nm (He-Ne laser)

3. การเลี้ยวเบนผ่านช่องเดี่ยว (Single-Slit / Fraunhofer Diffraction)

เมื่อแสงเลเซอร์ผ่านช่องแคบกว้าง a จะเกิด pattern แถบสว่าง-มืดบนจอ (ถ้าจออยู่ไกลมากเทียบกับช่อง → Fraunhofer diffraction)

แถบกลาง (central maximum) สว่างที่สุดและกว้างที่สุด แถบข้างเคียง (secondary maxima) สว่างน้อยลงตามลำดับ

Fraunhofer Single-Slit Diffraction:
I(θ) = I₀ × [sin(β) / β]²
β = π a sin(θ) / λ

ตำแหน่ง Minima (แถบมืด):
a sin(θ) = mλ   (m = ±1, ±2, ±3, ...)

ตำแหน่งบนจอ (small angle):
y_m = mλL / a

ข้อสังเกต:
• ช่องแคบลง (a ลด) → pattern กว้างขึ้น
• ช่องกว้างขึ้น (a เพิ่ม) → pattern แคบลง
• จอไกลขึ้น (L เพิ่ม) → pattern กว้างขึ้น

4. เกรตติงเลี้ยวเบน (Diffraction Grating)

เกรตติงมีช่องจำนวนมาก (เช่น 5,276 ช่อง/cm) ห่างกันเท่าๆ กัน เมื่อแสงผ่านเกรตติง จะเกิดจุดสว่างแคบๆ (sharp maxima) ที่ตำแหน่งตามเงื่อนไข constructive interference

Grating Equation (สมการเกรตติง):
d sin(θ) = mλ   (m = 0, ±1, ±2, ...)

คำนวณ λ จากข้อมูลทดลอง:
λ = d × x / L

d = ระยะห่างระหว่าง slits = 1/(จำนวน grooves ต่อ cm) เช่น
  5,276 grooves/cm → d = 1.896 × 10⁻⁴ cm = 1.896 μm
  13,400 grooves/inch = 5,276 grooves/cm

x = ระยะจาก central max ถึง 1st order max (วัดบนจอ)
L = ระยะจากเกรตติงถึง 1st order max บนจอ

ค่าอ้างอิง:
λ(He-Ne) = 6.328 × 10⁻⁷ m = 632.8 nm
% Error = |λ_measured − 632.8 nm| / 632.8 nm × 100
