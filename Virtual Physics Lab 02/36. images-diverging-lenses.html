วัตถุประสงค์

สังเกตลักษณะของภาพที่เกิดจากเลนส์เว้า (diverging lens / double concave lens) และเปรียบเทียบกับเลนส์นูน โดยแบ่งเป็น 2 ส่วน:

  • Part A: หาความยาวโฟกัสของเลนส์เว้า ด้วยวิธีวัดลำแสงเลเซอร์
  • Part B: ประกอบกล้องโทรทรรศน์กาลิเลโอ (Galileo Telescope) จากเลนส์นูน + เลนส์เว้า

อุปกรณ์

  • เลนส์เว้า (double concave lens) — ทางยาวโฟกัสประมาณ −5 ถึง −15 cm
  • เลนส์นูน (double convex lens) ทางยาวโฟกัสมาก (เช่น 30–50 cm) — ใช้ใน Part B
  • เลเซอร์กำลังต่ำ (ฮีเลียม-นีออน หรือ LED laser) ปลอดภัยต่อดวงตา
  • กระดาษสมุด/กระดาษขาว สำหรับรับแสง
  • ม้านั่งเชิงแสง (optical bench) หรือไม้เมตร
  • ตัวยึดเลนส์ 2 อัน, ตัวยึดกระดาษ
  • โฟโตมิเตอร์ (photometer) — เสริม หากต้องการวัดความเข้มแสง
⚠️ ความปลอดภัย: ห้ามมองเข้าไปในลำเลเซอร์โดยตรง แม้เป็นกำลังต่ำ อาจทำให้จอประสาทตาเสียหายได้

Part A — หาทางยาวโฟกัสของเลนส์เว้า

หลักการ

ลำแสงเลเซอร์ (รังสีขนานกันแน่น) เมื่อผ่านเลนส์เว้าจะกระจายออก แสงเคลื่อนที่ออกจากเลนส์ราวกับมาจากจุดโฟกัสเสมือน F ซึ่งอยู่ด้านเดียวกับแหล่งกำเนิดแสง (ด้านเลเซอร์)

ถ้าวัดเส้นผ่านศูนย์กลางของวงแสง D ที่ระยะ X จากเลนส์ต่างๆ จะได้ความสัมพันธ์เชิงเส้น:

D = D₀ + (D₀/|f|) · X

เมื่อเขียนกราฟ D (แกน y) กับ X (แกน x) แล้วต่อเส้นไปทางซ้าย (extrapolation) จนตัดแกน X ที่จุด D=0 ค่า X ที่จุดตัดนั้นคือ −|f| จึงได้ f = −|f| (เลนส์เว้ามีค่าเป็นลบ)

วิธีการทดลอง

  1. วางเลเซอร์กำลังต่ำบนโต๊ะ — ระวังไม่ให้ชี้ไปที่ดวงตาของใคร
  2. เปิดเลเซอร์ ส่องลำแสงผ่านเลนส์เว้า จะสังเกตเห็นลำแสงแผ่ออกเป็นวงกลมกว้างขึ้น
  3. ถือกระดาษสมุดชนตัวเลนส์ (X = 0 cm โดยประมาณ) ใช้ดินสอร่างขอบวงแสงที่เห็น
  4. เลื่อนกระดาษออกจากเลนส์ทุก 5 cm (5, 10, 20, 30, 40, 50, 60 cm) ร่างขอบวงแสงทุกตำแหน่ง
  5. วัดเส้นผ่านศูนย์กลาง D ของวงแสงแต่ละระยะ บันทึกลงตาราง
  6. พลอตกราฟ D (แกน y, cm) vs X (แกน x, cm)
  7. ลากเส้นตรงผ่านจุดข้อมูลให้ดีที่สุด (best fit line) แล้วต่อเส้นไปทางซ้ายจนตัดแกน X
  8. อ่านค่า X ที่จุดตัด = −|f| → ได้ทางยาวโฟกัสของเลนส์เว้า
💡 หมายเหตุ: นักเรียนจะได้ข้อมูลจริงคล้ายตารางตัวอย่าง เช่น X = 5,10,20,30,40,50,60 cm → D = 1.3, 2.3, 4.7, 8.0, 11.0, 13.1, 16.4 cm
(ตัวเลขจะแตกต่างกันไปตามเลนส์ที่ใช้ — slope = D₀/|f|)

วิธีการใน Simulation (Part A)

  1. ในแท็บ การทดลอง เลือก Part A
  2. ลากกระดาษบนม้านั่งเชิงแสง (ระยะ X = 5, 10, 20, ... 60 cm)
  3. อ่านขนาดวงแสง D จากกล่อง "วงแสงบนกระดาษ" มุมขวาบน — หรือใช้ ไม้บรรทัด (↕) วัดตรงๆ บนวงแสง
  4. กรอก X และ D ลงในช่อง "บันทึกค่า" แล้วกดปุ่ม "เพิ่มจุด"
  5. ทำซ้ำอย่างน้อย 5–7 จุด จนได้กราฟเส้นตรง
  6. ระบบจะ fit เส้นตรง และ extrapolate ให้อัตโนมัติ — อ่านค่า f จากจุดตัดแกน X
  7. เปรียบเทียบกับ f จริง — ถ้ายังไม่ดี กด "สุ่มเลนส์ใหม่" ทำซ้ำ

Part B — กล้องโทรทรรศน์กาลิเลโอ (Galileo Telescope)

หลักการ

กาลิเลโอค้นพบว่าถ้านำเลนส์นูนทางยาวโฟกัสยาว (F₁ = objective) + เลนส์เว้าทางยาวโฟกัสสั้น (F₂ = ocular/eyepiece) มาวางในแนวเดียวกัน จะได้กล้องโทรทรรศน์ที่ให้ภาพตั้งตรง (ไม่กลับหัวเหมือนกล้องเคปเลอร์)

M = −F₁/F₂

เนื่องจาก F₂ เป็นลบ (เลนส์เว้า) ค่า M จึงเป็นบวก → ภาพตั้งตรง

ระยะห่างระหว่างเลนส์ทั้งสองในกล้องกาลิเลโอ คือ L = F₁ + F₂ (เป็นบวก เพราะ |F₁| > |F₂|)

วิธีการทดลอง

  1. ค้นหาทางยาวโฟกัสของเลนส์เว้าด้วยวิธี Part A → ได้ F₂ (เป็นลบ)
  2. หาทางยาวโฟกัสของเลนส์นูนด้วยวิธี Lab 35 (แสงอาทิตย์) → ได้ F₁ (ยาว เช่น 30–50 cm)
  3. วางเลนส์นูนห่างจากเลนส์เว้าเป็นระยะ L = F₁ + F₂
  4. มองผ่านเลนส์เว้า (ด้านที่ใกล้ตา) ไปยังวัตถุที่อยู่ไกล เช่น ป้ายหน้าห้อง, อาคารฝั่งตรงข้าม, ต้นไม้
  5. สังเกตว่าภาพที่เห็น:
    • ขยายหรือเล็กลง?
    • ตั้งตรงหรือกลับหัว?
    • คำนวณ M = −F₁/F₂ เปรียบเทียบกับขนาดภาพที่เห็น
  6. บันทึกผลการสังเกต

วิธีการใน Simulation (Part B)

  1. เลือก Part B: กล้องกาลิเลโอ
  2. ลากเลนส์นูน (F₁) และเลนส์เว้า (F₂) บนม้านั่ง — ระบบจะให้ลากตำแหน่งแต่ละเลนส์
  3. ลากดวงตา 👁 ให้ใกล้เลนส์เว้า
  4. ดูกล่อง "มุมมองผ่านกล้อง" มุมขวาบน เพื่อเห็นภาพของวัตถุไกล (เช่น ต้นไม้/บ้าน)
  5. ปรับตำแหน่งเลนส์จนภาพคมชัด → L = F₁ + F₂
  6. กรอกค่า F₁ (บวก) และ F₂ (ลบ) ลงในช่อง แล้วกด "บันทึก"
  7. ระบบคำนวณกำลังขยาย M = −F₁/F₂ ให้เปรียบเทียบ

คำถามหลังการทดลอง

  1. ทำไมจึงหาทางยาวโฟกัสของเลนส์เว้าด้วยวิธี Part A (ใช้แสงอาทิตย์) ไม่ได้?
    → เพราะเลนส์เว้ากระจายแสง ไม่รวมเป็นจุดโฟกัส ทำให้ไม่มีภาพจริงบนกระดาษ
  2. ภาพที่เห็นผ่านเลนส์เว้าเมื่อมองวัตถุเทียบกับขนาดจริง เป็นอย่างไร?
    → เล็กกว่า, ตั้งตรง, เสมือน (อยู่ด้านเดียวกับวัตถุ)
  3. ทำไมกล้องกาลิเลโอให้กำลังขยายเป็นบวก?
    → เพราะ F₂ เป็นลบ → M = −F₁/F₂ จึงเป็นบวก = ภาพตั้งตรง

1. รังสีหลัก 3 เส้นผ่านเลนส์เว้า

เลนส์เว้า (diverging lens / double concave) หักเหแสงให้กระจายออก เราใช้รังสีหลัก 3 เส้นจากปลายวัตถุเพื่อหาตำแหน่งภาพ:

  • รังสี 1 (แดง): ขนานแกนหลัก → ผ่านเลนส์ → หักเหกระจายออก ราวกับมาจากจุดโฟกัสเสมือน F (ด้านเดียวกับวัตถุ)
  • รังสี 2 (เขียว): ผ่านจุดศูนย์กลางเลนส์ → ตรงไปไม่หักเห
  • รังสี 3 (น้ำเงิน): มุ่งไปยังจุดโฟกัส F' (ด้านหลัง) → ผ่านเลนส์ → ออกขนานแกนหลัก

จุดที่ ส่วนต่อของรังสี (เส้นประ) ตัดกัน คือ ตำแหน่ง ภาพเสมือน (อยู่ด้านเดียวกับวัตถุเสมอ)

2. ภาพจากเลนส์เว้าเป็น "เสมือน ตั้งตรง เล็กลง" เสมอ

ไม่ว่าวัตถุจะอยู่ที่ระยะใดก่อนเลนส์เว้า ภาพที่ได้จะเป็น:

ระยะวัตถุ doระยะภาพ diชนิดทิศทางขนาดM
ใกล้เลนส์ (do < |f|)− (|di| < do)เสมือนตั้งตรงเล็กกว่า0 < M < 1
do = |f||f|/2เสมือนตั้งตรงครึ่งหนึ่งM = 0.5
ไกลเลนส์ (do > |f|)− (|di| < |f|)เสมือนตั้งตรงเล็กกว่ามากM → 0

ภาพจากเลนส์เว้า จับภาพบนจอไม่ได้ — เพราะรังสีจริงไม่ตัดกัน (ต้องต่อเส้นรังสีย้อนกลับจึงจะตัด)

3. สมการเลนส์บางกับเลนส์เว้า

1/f = 1/do + 1/di

สำหรับเลนส์เว้า f < 0 (f เป็นลบ) ทำให้ 1/di < 0 → di < 0 เสมอ → ภาพเสมือน

ตัวอย่างคำนวณ: เลนส์เว้า f = −10 cm, วัตถุอยู่ที่ do = 20 cm

1/di = 1/f − 1/do = −1/10 − 1/20 = −3/20
di = −20/3 ≈ −6.67 cm (ภาพอยู่ด้านหน้าเลนส์ 6.67 cm)
M = di/do = −6.67/20 ≈ −0.33 (|M| < 1 → เล็กกว่า)

4. วิธีวัดโฟกัสด้วยลำแสงเลเซอร์ (Beam Expansion Method)

เลเซอร์ให้ลำแสงขนาน เมื่อผ่านเลนส์เว้า ลำแสงจะแผ่ออกเป็นวงกลม ราวกับแสงกระจายจากจุดโฟกัสเสมือน F ด้านหลังเลนส์ (ด้านเดียวกับเลเซอร์)

จากรูปสามเหลี่ยมคล้ายกัน:

D / (|f| + X) = D₀ / |f|

จัดรูปใหม่ได้:

D = D₀ + (D₀/|f|) · X

กราฟ D vs X เป็นเส้นตรง:

  • ความชัน (slope) = D₀/|f|
  • y-intercept = D₀ (เส้นผ่านศูนย์กลางลำแสงที่เลนส์)
  • x-intercept = −|f| ← ต่อเส้นไปทางซ้าย จุดตัดให้ค่าโฟกัส

5. กล้องโทรทรรศน์กาลิเลโอ (Galileo Telescope)

ประกอบด้วยเลนส์ 2 ตัว วางในแนวเดียวกัน:

  • Objective: เลนส์นูนทางยาวโฟกัสยาว F₁ (e.g. 40 cm) — อยู่ด้านวัตถุ
  • Ocular / Eyepiece: เลนส์เว้าทางยาวโฟกัสสั้น F₂ (e.g. −8 cm) — อยู่ด้านตา
  • ระยะห่างระหว่างเลนส์: L = F₁ + F₂
M = −F₁ / F₂

เนื่องจาก F₂ < 0 (เลนส์เว้า) ค่า M จึงเป็นบวก → ภาพตั้งตรง (ไม่กลับหัว)

ทำไมภาพถึงตั้งตรง?

  1. แสงจากวัตถุไกล (มาถึงเลนส์แบบขนาน) ผ่านเลนส์นูน F₁ → ควรจะรวมเป็นภาพจริงกลับหัวที่ระยะ F₁ จากเลนส์นูน
  2. แต่ก่อนจะรวมเป็นภาพจริง แสงเจอเลนส์เว้า F₂ ก่อน (วางอยู่ห่างจากเลนส์นูน F₁ + F₂)
  3. เลนส์เว้าทำให้แสงที่กำลังลู่เข้าหากัน "กลับมาเป็นขนาน" ออกสู่ดวงตา — ตา focus บนภาพเสมือนที่ระยะอนันต์
  4. เพราะแสงไม่ได้ตัดกันจริงก่อนถึงเลนส์เว้า ภาพจึง ไม่พลิกกลับ → เห็นตั้งตรง ✨
🔭 การใช้งานจริง: กล้องส่องละครโอเปร่า (opera glasses), กล้องดูนก รุ่นแรกๆ ใช้ระบบกาลิเลโอ — กะทัดรัด และเห็นภาพตั้งตรง แต่กำลังขยายต่ำและมุมมอง (FOV) แคบ
