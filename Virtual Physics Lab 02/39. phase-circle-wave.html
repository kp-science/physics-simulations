💡 ฐาน B · เฟสริบบิ้น (Ribbon Phase)

ตรวจสอบ misconception M3.3 (เฟสไม่มีความหมาย ผิด!) · M3.4 (180° ไม่เกี่ยวข้อง ผิด!)

1เลือกฐาน B · จะเห็นริบบิ้น 4 เส้นที่ Δx = λ/4, λ/2, 3λ/4, λ พร้อมวงกลมอ้างอิง (phasor) ทางซ้ายบน
2กด ▶ เริ่ม · สังเกตลูกบอลบนวงกลมหมุนไปเรื่อย ๆ เส้นประแนวนอนจะลาก "ภาพฉาย y" ไปที่กราฟ y–t ทางขวา
3สังเกตริบบิ้นแต่ละเส้น ว่าสั่น พร้อม หรือ สวน กับริบบิ้นแดง (R ที่ต้น):
  • Δx = λ/4 → ช้าไป 90° (quadrature)
  • Δx = λ/2 → สวนทาง 180° (anti-phase)
  • Δx = 3λ/4 → ช้าไป 270°
  • Δx = λ → พร้อมกัน 360° = 0° (in-phase)
4กรอกตาราง ว่าแต่ละริบบิ้น "พร้อม" หรือ "สวน" หรือ "ช้า/เร็ว กี่องศา" เทียบกับ R

📐 ทฤษฎีและคำอธิบาย

1) การเคลื่อนที่วงกลมสม่ำเสมอ ↔ SHM ↔ คลื่น

จุดที่หมุนรอบวงกลมรัศมี A ด้วยอัตราเร็วเชิงมุม ω คงที่ ภาพฉายในแนวดิ่ง y = A sin(ωt) คือ การเคลื่อนที่แบบฮาร์มอนิกอย่างง่าย (SHM) · ถ้าเอา SHM นี้มาเลื่อนไปตามแกน x ด้วยความเร็วคลื่น v = fλ จะได้คลื่นต่อเนื่อง:

y(x, t) = A sin(ωt − kx + φ₀)

ภาพ: วงกลมอ้างอิง → ภาพฉาย y → กราฟ y–t (ดูแอนิเมชันหมุนต่อเนื่อง)

2) เฟส φ คืออะไร

เฟส คือ "ตำแหน่งบนวงกลม" ของจุดสั่น ณ เวลา t หนึ่ง ๆ — เป็นมุมหมุน หน่วย rad หรือ ° บอกว่าตอนนี้วัตถุสั่นอยู่ช่วงไหนของรอบ (ขึ้น/ลง/สูงสุด/ต่ำสุด)

  • φ = 0° → y = 0 กำลังขึ้น
  • φ = 90° → y = +A (สูงสุด)
  • φ = 180° → y = 0 กำลังลง
  • φ = 270° → y = −A (ต่ำสุด)
  • φ = 360° → กลับมา y = 0 ครบ 1 รอบ

3) กราฟ y–t vs y–x (อย่าสับสน!)

กราฟ y–t (ที่ตำแหน่งเดียว)

จับตาที่จุด จุดเดียว (เช่น R) แล้วดูว่า y ของมันเปลี่ยนตามเวลาอย่างไร · คาบที่เห็น = T (คาบเวลา)

y(t) = A sin(ωt)

กราฟ y–x (เวลาเดียว)

ถ่ายภาพเชือก ชั่วขณะเดียว (snapshot) แล้วดูรูปร่าง y ตามตำแหน่ง x · คาบที่เห็น = λ (ความยาวคลื่น)

y(x) = A sin(−kx)

4) ความสัมพันธ์ Δφ ↔ Δx (หัวใจของแผนนี้)

ถ้า 2 จุดอยู่ห่างกัน Δx บนคลื่นต่อเนื่อง ความแตกต่างของเฟสคือ

Δφ = 2π · (Δx / λ) = 360° · (Δx / λ)

ภาพ: ริบบิ้น 4 เส้นที่ Δx = λ/4, λ/2, 3λ/4, λ พร้อมวงกลมอ้างอิงแสดงมุมเฟส

ΔxΔφ (rad)Δφ (องศา)สั่นสัมพันธ์กับ R
λ/4π/290°ช้าไป 1/4 รอบ (quadrature)
λ/2π180°สวนทาง (anti-phase)
3λ/43π/2270°ช้าไป 3/4 รอบ
λ2π360° = 0°พร้อมกัน (in-phase)

5) Pulse vs Continuous — ความเร็วเท่ากันหรือไม่?

คำตอบ: เท่ากัน ในตัวกลางเดียวกัน · ความเร็วคลื่นตามขวางในเชือกคือ
v = √(T / μ)

โดย T = แรงตึงเชือก, μ = มวลต่อความยาว · v ขึ้นกับสมบัติตัวกลางเท่านั้น · ไม่ขึ้นกับว่าเป็นดล (1 ลูก) หรือ ต่อเนื่อง (สั่นซ้ำ) · ทั้งคู่วิ่งเร็วเท่ากัน · นี่เป็น misconception M3.2 ที่นักเรียนมักเข้าใจผิด

6) การซ้อนคลื่น (Superposition) และ Δφ = 180°

เมื่อคลื่น 2 ลูกมีความถี่และแอมพลิจูดเท่ากัน แต่ต่างเฟสกัน Δφ ผลรวมคือ

y₁ + y₂ = 2A cos(Δφ/2) · sin(ωt − kx + Δφ/2)

ดังนั้น แอมพลิจูดผลรวม = 2A cos(Δφ/2):

  • Δφ = 0° → 2A · cos(0) = 2A (เสริมสูงสุด — constructive)
  • Δφ = 90° → 2A · cos(45°) ≈ 1.41A
  • Δφ = 180° → 2A · cos(90°) = 0 (หักล้างหมด — destructive)
  • Δφ = 360° → 2A · cos(180°) = −2A (= +2A แอมป์สูงสุด in-phase อีกรอบ)

7) Misconceptions ที่ตรวจสอบ

M3.1 "ดล = ต่อเนื่อง" — ผิด · ดลคือสะบัด 1 ลูก ส่วนต่อเนื่องคือสั่นซ้ำ ๆ ได้สเปกตรัมคลื่นต่อเนื่องหลายลูก
M3.2 "v ดล ≠ v ต่อเนื่อง" — ผิด · v ขึ้นกับตัวกลาง ไม่ได้ขึ้นกับชนิด source
M3.3 "เฟสไม่มีความหมายกายภาพ" — ผิด · เฟสบอกตำแหน่งบนวงกลมอ้างอิง = สถานะการสั่น ณ ขณะนั้น
M3.4 "180° ไม่เกี่ยวข้อง" — ผิด · 180° คือจุดที่การซ้อนคลื่นหักล้างพอดี เป็นหัวใจของ noise-canceling, interference
