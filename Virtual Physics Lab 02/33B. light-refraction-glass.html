🧪 อุปกรณ์การทดลอง (Real Lab)

  • กล่องแก้วสี่เหลี่ยม (glass square) 7 × 7 ซม. หรือกล่องใส่ของเหลว
  • เข็มหมุดตรง 4 ตัว
  • กระดาษ A4 + แผ่นกระดาษแข็ง
  • ไม้โปรแทรกเตอร์
  • ไม้บรรทัด

📝 ขั้นตอนการทดลอง (Real Lab)

  1. วางกล่องแก้วบนกระดาษแข็ง ให้ขอบด้านเรียบตรงกับเส้นที่กำหนดไว้
  2. ปักเข็มหมุด P₁ และ P₂ ด้านบน (ด้านที่แสงตกกระทบ) ห่างกัน 3–4 ซม.
  3. ก้มมองจากอีกฝั่ง (ด้านล่าง) ให้ตาอยู่ระดับเดียวกับแก้ว
  4. ปักเข็มหมุด P₃ และ P₄ ด้านล่าง ให้ตรงแนวกับ P₁ P₂ ที่มองผ่านแก้ว
  5. ยกแก้วออก ลากเส้นผ่าน P₁P₂ (รังสีตกกระทบ) และ P₃P₄ (รังสีออก)
  6. ลากเส้นเชื่อมจุดที่รังสีเข้าและออกจากแก้ว → รังสีหักเหภายใน
  7. ลากเส้น Normal ที่จุดเข้าแก้ว
  8. วัดมุม θ₁ (ในอากาศ) และ θ₂ (ในตัวกลาง) ด้วยโปรแทรกเตอร์
  9. คำนวณ n = sinθ₁ / sinθ₂ แล้วทำซ้ำอย่างน้อย 5 มุม

💻 วิธีใช้ Simulation — โหมดบันทึกค่าเอง

  1. ขั้นตอน 1: คลิกเหนือกล่องแก้ว วาง P₁ แล้ว P₂
  2. ขั้นตอน 2: ดวงตาปรากฏด้านล่าง — ลากดวงตาไปมาแล้วดูกล่อง "มุมมองของตา" มุมขวาบน → จะเห็นเข็มหมุด P₁ P₂ ผ่านแก้ว (ตำแหน่งที่เห็นจะเลื่อนเนื่องจากการหักเห) → คลิกวาง P₃ แล้ว P₄ ใต้กล่องแก้วให้ตรงแนวกับ P₁ P₂ ที่มองผ่านแก้ว
  3. ยืนยัน: กด ✓ เสร็จสิ้น
  4. ขั้นตอน 3: รังสีจะปรากฏ: P₁→P₂→จุดเข้าแก้ว→รังสีหักเหในแก้ว→จุดออกแก้ว→P₃→P₄ พร้อมเส้น Normal ที่ทุกรอยต่อ
  5. ขั้นตอน 4: เปิด 📐 โปรแทรกเตอร์ ลาก+หมุนวัดมุม θ₁ (ในอากาศ) และ θ₂ (ในตัวกลาง) ที่จุดเข้าแก้ว → กรอกในตาราง

📊 สูตรคำนวณ

n = sinθ₁ / sinθ₂ (เมื่อ n₁ = 1 สำหรับอากาศ)
%Error = |nวัดได้ − nจริง| ÷ nจริง × 100
ความเร็วแสงในตัวกลาง: v = c / n = (3.0 × 10⁸) / n m/s

💡 กฎของสเนลล์ (Snell's Law)

เมื่อแสงเดินทางจากตัวกลางหนึ่ง (เช่น อากาศ) ไปยังอีกตัวกลางหนึ่งที่หนาแน่นกว่า แสงจะ เบี่ยงเข้าหาเส้น Normal

n₁ sin θ₁ = n₂ sin θ₂
  • θ₁ = มุมตกกระทบ (ในอากาศ) — วัดจากเส้น Normal
  • θ₂ = มุมหักเห (ในตัวกลาง) — วัดจากเส้น Normal

🌊 ทำไมแสงถึงหักเห? — Wavefront

Wavefront ด้านที่เข้าตัวกลางก่อน ช้าลง → wavefront หมุนเปลี่ยนทิศ → รังสีเบี่ยงเข้าหา Normal

d / d' = vอากาศ / vตัวกลาง = n

🔺 การสะท้อนกลับหมด (Total Internal Reflection)

เมื่อแสงเดินทางจากตัวกลางหนาแน่น → ตัวกลางเบา ถ้า θ₁ ≥ θc แสงจะสะท้อนกลับหมด

sin θc = 1 / n
