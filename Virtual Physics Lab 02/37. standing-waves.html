จุดประสงค์

สร้างคลื่นนิ่ง (Standing Wave) บนเส้นเชือกที่สั่นด้วยความถี่คงที่ และเรียนรู้การหาความยาวคลื่น (λ) จากการวัดระยะห่างระหว่าง node ที่อยู่ติดกัน

📐 ทฤษฎี — คลื่นนิ่ง (Standing Waves)

1. คลื่นนิ่งคืออะไร?

คลื่นนิ่ง (Standing Wave) เกิดจากการซ้อนทับ (superposition) ของคลื่น 2 ขบวนที่มีความถี่ ความยาวคลื่น และแอมพลิจูดเท่ากัน แต่เคลื่อนที่สวนทางกัน ผลลัพธ์คือรูปแบบคลื่นที่ดูเหมือนอยู่กับที่ — บางจุดสั่นแรงมาก (antinode) บางจุดไม่สั่นเลย (node)

2. Node และ Antinode

Node คือจุดที่ไม่มีการสั่น (amplitude = 0) เกิดจากการแทรกสอดแบบหักล้าง (destructive interference)
Antinode คือจุดที่สั่นมากที่สุด (amplitude สูงสุด) เกิดจากการแทรกสอดแบบเสริม (constructive interference)

ระยะระหว่าง node ที่อยู่ติดกัน = λ/2

ความสัมพันธ์ระหว่าง Node กับ λ
d = ระยะระหว่าง node ติดกัน = λ/2
λ = 2d
จำนวน loop (n) = 2L/λ = L/d

3. ความเร็วคลื่นบนเส้นเชือก

สูตรความเร็วคลื่นบนเส้นเชือก
v = √(T/μ)
โดย T = แรงตึงเชือก (tension) = น้ำหนัก W = mg (N)
      μ = มวลต่อความยาว (linear density) (kg/m)

v = fλ
ดังนั้น λ = v/f = √(T/μ) / f

4. เงื่อนไข Resonance

คลื่นนิ่งจะเกิดขึ้นอย่างชัดเจน (resonance) เมื่อความยาวเชือก L เป็นจำนวนเต็มของ λ/2:

เงื่อนไข Resonance (ปลายทั้งสองตรึง)
L = nλ/2   (n = 1, 2, 3, ...)
λₙ = 2L/n
fₙ = nv/(2L) = (n/2L)√(T/μ)

n = 1 → Fundamental (1st harmonic) — 1 loop
n = 2 → 2nd harmonic — 2 loops
n = 3 → 3rd harmonic — 3 loops

5. ผลของการเปลี่ยน Tension

เมื่อเพิ่มน้ำหนัก W ที่แขวนที่ปลายเชือก → T เพิ่ม → v = √(T/μ) เพิ่ม → λ = v/f เพิ่ม → จำนวน loop n = 2L/λ ลดลง

ตัวอย่างการคำนวณ
กำหนด: f = 60 Hz, L = 1.50 m, μ = 4.0 g/m = 0.004 kg/m

Trial 1: W = 0.029 N
  v = √(0.029/0.004) = √7.25 = 2.69 m/s
  λ = 2.69/60 = 0.0449 m
  n = 2(1.50)/0.0449 = 66.8 ≈ 67 loops

Trial 2: W = 0.44 N
  v = √(0.44/0.004) = √110 = 10.49 m/s
  λ = 10.49/60 = 0.175 m
  n = 2(1.50)/0.175 = 17.1 ≈ 17 loops

Trial 3: W = 1.32 N
  v = √(1.32/0.004) = √330 = 18.17 m/s
  λ = 18.17/60 = 0.303 m
  n = 2(1.50)/0.303 = 9.9 ≈ 10 loops
