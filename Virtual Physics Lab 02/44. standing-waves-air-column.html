เตรียมอุปกรณ์

ตามภาพอ้างอิงต้องเตรียม:

  • หลอดพลาสติก (ยาว ≈ 0.5 m) เช่น หลอดเก็บไม้กอล์ฟ — ปลายเปิดทั้งสองข้าง
  • กระบอกแก้ว (ยาว ≈ 0.5 m) ปิดปลายข้างหนึ่ง บรรจุน้ำ สูงพอเหมาะ
  • ไม้บรรทัด/ตลับเมตร — วัดความยาวคอลัมน์อากาศ
  • ส้อมเสียง (tuning forks) ความถี่หลากหลายระหว่าง 256–512 Hz
📝 ใน simulation นี้: หลอดแสดงด้วยรูปทรงกระบอกใส · ผิวน้ำคือปลายปิดด้านล่าง · ปลายเปิดด้านบนคือที่วางส้อมเสียง
2

ตั้งค่าและเริ่มสั่นส้อมเสียง

  1. เลือกความถี่ส้อมเสียงในเมนู (เริ่มที่ 512 Hz)
  2. ปรับ อุณหภูมิอากาศในห้อง ให้ตรงกับสภาพจริง (15–30 °C)
  3. กด "🔔 เคาะส้อมเสียง" — เสียงจะค่อย ๆ จางลงใน ~3 วินาที (เหมือนการเคาะจริง)
  4. หรือใช้ "🎵 ต่อเนื่อง" เพื่อให้เสียงดังอยู่ตลอด (เหมาะกับการกวาดหาตำแหน่งเรโซแนนซ์)
⚠️ หากใช้ส้อมเสียงจริง ต้องเคาะกับยางหรือลูกบอลยาง อย่าเคาะกับโต๊ะแข็งเพราะอาจทำให้ส้อมเสียงเสีย
3

หาความยาวที่เสียงดังที่สุด (Manual Mode)

  1. ลากหลอดขึ้น-ลงในกระบอกน้ำ — คลิกค้างบน canvas แล้วเลื่อนเมาส์
  2. สังเกตแถบ loudness ที่มุมล่างซ้าย — เสียงดังที่สุดเมื่อแถบเต็มและเปลี่ยนเป็นสีแดง
  3. เมื่อเสียงดังสุด กด "💾 บันทึก L ตอนนี้" — โปรแกรมจะบันทึกความยาวอากาศเหนือน้ำในช่อง L₁
  4. ลากหลอดต่อไปจนเจอตำแหน่ง เรโซแนนซ์ถัดไป (เสียงดังขึ้นอีกครั้ง) — บันทึกเป็น L₂
  5. โปรแกรมจะคำนวณ λ = 2(L₂ − L₁) และ v = f·λ อัตโนมัติ
  6. เปลี่ยนความถี่ส้อมเสียงและทำซ้ำขั้นตอน 3–5 อย่างน้อย 2–3 ความถี่
📏 เหตุใดต้องวัด 2 ตำแหน่ง? เพราะการวัด L1 = λ/4 เพียงค่าเดียวมี end-correction (e ≈ 0.6r) ทำให้ค่าคลาดเคลื่อน แต่การวัดผลต่าง L2 − L1 = λ/2 ทำให้ end-correction ตัดกัน
4

โหมด Auto Scan

หากนักเรียนต้องการเปรียบเทียบหรือตรวจสอบผลของตนเอง:

  1. กดปุ่ม "🤖 เริ่ม Auto Scan"
  2. โปรแกรมจะเลื่อนหลอดอย่างช้า ๆ จากบนลงล่าง พร้อมวัดความดังทุกตำแหน่ง
  3. เมื่อพบจุดยอด (peak) ของความดัง 2 จุดแรก จะบันทึกใส่ตารางอัตโนมัติ
  4. กด "🛑 หยุด Auto Scan" เพื่อหยุดกลางคัน
⚠️ Auto Mode เหมาะใช้หลังนักเรียนทำ Manual Mode แล้ว — เพื่อเทียบความแม่นยำของการวัดของตัวเอง

1. คลื่นนิ่งในหลอดปลายปิดข้างเดียว (Closed-end pipe)

กระบอกในการทดลองนี้มี ผิวน้ำเป็นปลายปิด ด้านล่างและ ปลายเปิด ด้านบน เมื่อส้อมเสียงสั่นเหนือปากหลอด คลื่นเสียงจะเดินทางลงไปสะท้อนที่ผิวน้ำกลับขึ้นมาซ้อนทับกับคลื่นตกกระทบ เกิดเป็น คลื่นนิ่ง (standing wave)

  • ที่ผิวน้ำ (ปลายปิด): โมเลกุลอากาศเคลื่อนที่ไม่ได้ → displacement node (จุดบัพของการกระจัด) แต่เป็น pressure antinode
  • ที่ปากหลอด (ปลายเปิด): โมเลกุลอากาศสั่นได้เต็มที่ → displacement antinode (จุดปฏิบัพของการกระจัด)
  • เมื่อปากหลอดเป็น antinode พอดี → เกิดเรโซแนนซ์ (เสียงดังที่สุด)

ความยาวอากาศที่ทำให้เกิดเรโซแนนซ์:

Ln = (2n − 1) · λ/4, n = 1, 2, 3, ...

ระยะระหว่าง 2 ตำแหน่งเรโซแนนซ์ติดกัน:

ΔL = Ln+1 − Ln = λ/2

→ จึงคำนวณ λ = 2·ΔL = 2(L₂ − L₁)

n=1
L = λ/4
n=2
L = 3λ/4
n=3
L = 5λ/4
n=4
L = 7λ/4

2. ความสัมพันธ์ v = f·λ

คลื่นเสียงทุกชนิดเชื่อม ความถี่ กับ ความยาวคลื่น ผ่านอัตราเร็วของเสียงในตัวกลาง:

v = f · λ

กราฟแสดงคลื่นนิ่ง (displacement amplitude vs ตำแหน่ง):

เส้นทึบ: แอมพลิจูดของการกระจัด · เส้นประ: คลื่นเสียงในขณะหนึ่ง · จุดสีแดง: antinode · จุดสีเขียว: node

3. อัตราเร็วของเสียงในอากาศกับอุณหภูมิ

อัตราเร็วของเสียงในอากาศแห้งขึ้นกับอุณหภูมิ (T เป็น °C):

v(T) = 331.4 + 0.6·T (m/s)

เช่น ที่ T = 22 °C: v ≈ 331.4 + 13.2 = 344.6 m/s

หรือสูตรที่แม่นกว่า (อุณหภูมิสัมบูรณ์ K): v = 331.4·√(TK/273.15)

4. End Correction (e)

จริง ๆ แล้ว antinode ของอากาศไม่ได้อยู่ตรงปากหลอดพอดี แต่อยู่เหนือปากเล็กน้อย เป็นระยะ:

e ≈ 0.6 · r (r = รัศมีหลอด)

ดังนั้นความยาวเรโซแนนซ์ตำแหน่งแรก L1 + e = λ/4

เหตุผลที่ต้องวัด 2 ตำแหน่ง: เมื่อนำ L2 − L1 มาลบกัน end-correction e จะตัดออก ทำให้ค่า λ ที่ได้แม่นยำขึ้นมาก

(L2 + e) − (L1 + e) = 3λ/4 − λ/4 = λ/2 ✓

5. ทำไมเสียงจึงดังขึ้นที่ตำแหน่งเรโซแนนซ์?

เมื่อความถี่ของส้อมเสียงตรงกับ ความถี่ธรรมชาติ ของหลอดอากาศ (resonant frequency) คลื่นที่สะท้อนกลับมาจะ เสริมกัน (constructive interference) อย่างต่อเนื่อง พลังงานสะสมในหลอดสูงขึ้น แอมพลิจูดของอากาศที่ปากหลอดเพิ่มขึ้นมาก → หูได้ยินเสียงดังขึ้น

ในทางตรงกันข้าม ถ้าความยาวหลอดไม่ตรงกับ (2n−1)λ/4 คลื่นที่สะท้อนกลับมาจะ หักล้าง กับคลื่นตกกระทบ → เสียงเบา

กราฟ Loudness vs L: เห็นยอดสูงเป็นช่วงสม่ำเสมอทุก ๆ λ/2 — นี่คือสัญญาณบอกว่าระยะระหว่างยอด = λ/2
