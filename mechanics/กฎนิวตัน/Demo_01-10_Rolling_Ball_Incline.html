⚙️ ฟิสิกส์ระนาบเอียง (Inclined Plane)

a  = g sin θ
x  = ½ a t²
v  = a t
a  = คงที่ (ตลอดการเคลื่อนที่)
g = ความเร่งเนื่องจากแรงโน้มถ่วง (m/s²)  |  θ = มุมเอียงของระนาบเทียบกับแนวราบ
สมมติลูกบอลเริ่มจากหยุดนิ่ง (u = 0) และไม่มีแรงเสียดทาน (frictionless)

📏 กฎเลขคี่ของ Galileo (Odd-Number Rule)

Δxₙ  =  ½ a Δt² (2n − 1)
สำหรับ Δt คงที่ อัตราส่วนระยะทางในแต่ละช่วงเวลาเป็นเลขคี่เสมอ:
Δx₁ : Δx₂ : Δx₃ : Δx₄ : Δx₅ : Δx₆  =  1 : 3 : 5 : 7 : 9 : 11
pattern ตำแหน่ง   x = ½a(nΔt)²   (a=2, Δt=1 s)
n=0 |←1→| |←—3—→| |←——5——→|

📈 รูปร่างกราฟ

กราฟรูปร่างความชัน / ความหมาย
x–t พาราโบลา (โค้งขึ้น) x = ½at² → ฟังก์ชัน t² ความชันเพิ่มขึ้น = v เพิ่ม
v–t เส้นตรงผ่านจุดกำเนิด ความชัน = a = คงที่  |  พื้นที่ใต้กราฟ = x
a–t เส้นราบ (Horizontal) a = g sin θ = const ตลอดเวลา
