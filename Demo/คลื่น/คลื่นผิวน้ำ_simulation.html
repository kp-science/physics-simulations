🌊 คลื่นผิวน้ำ คืออะไร?

คลื่นผิวน้ำ (Water Surface Waves) เกิดจากการรบกวนผิวน้ำ ทำให้อนุภาคของน้ำสั่นขึ้น-ลงรอบตำแหน่งสมดุล แล้วส่งพลังงานออกไปโดยรอบในรูปแบบคลื่น โดยน้ำไม่ได้เคลื่อนที่ตามคลื่น

คลื่นผิวน้ำจัดเป็น คลื่นตามขวาง (Transverse wave) — ทิศการสั่นของอนุภาคน้ำ (ขึ้น-ลง) ตั้งฉากกับทิศการเดินทางของคลื่น

📐 ปริมาณสำคัญของคลื่น

สัญลักษณ์ชื่อความหมายหน่วย SI
Aแอมพลิจูดระยะกระจัดสูงสุดจากตำแหน่งสมดุลm
fความถี่จำนวนรอบการสั่นต่อหนึ่งวินาทีHz (s⁻¹)
Tคาบเวลาสำหรับหนึ่งรอบการสั่นs
λความยาวคลื่นระยะห่างระหว่างสันคลื่นสองสันติดกันm
vความเร็วคลื่นอัตราเร็วในการเดินทางของหน้าคลื่นm/s

🔣 สมการหลัก

v = f · λ (ความสัมพันธ์ความเร็ว-ความถี่-ความยาวคลื่น) T = 1 / f (ความสัมพันธ์คาบ-ความถี่) y(x,t) = A · sin(kx − ωt) (สมการคลื่นเดินทาง 1 มิติ) ค่าคงที่ที่เกี่ยวข้อง: k = 2π/λ (wave number, rad/m) ω = 2πf (angular frequency, rad/s)

⚠️ สมการนี้ใช้ได้กับคลื่นที่มีความเร็วคงที่และไม่มีการหน่วง

🔵🔵 การแทรกสอดของคลื่น (Wave Interference)

เมื่อคลื่น 2 ขบวนจากแหล่งกำเนิด S₁ และ S₂ เคลื่อนที่มาพบกัน จะเกิดการซ้อนทับ (Superposition):

y_รวม = A·sin(kr₁ − ωt) + A·sin(kr₂ − ωt + Δφ) โดย r₁, r₂ = ระยะจาก S₁, S₂ ไปยังจุดนั้น

ปฏิบัพ (Constructive) เกิดเมื่อ |r₁ − r₂| = 0, λ, 2λ, ... → คลื่นเสริมกัน แอมพลิจูดสูงสุด

บัพ (Destructive) เกิดเมื่อ |r₁ − r₂| = λ/2, 3λ/2, ... → คลื่นหักล้างกัน แอมพลิจูด = 0

〰 คลื่นนิ่ง (Standing Wave)

เกิดจากคลื่น 2 ขบวนที่มีแอมพลิจูดและความถี่เท่ากัน เดินทางสวนทางกัน:

y₁ = A·sin(kx − ωt) ← เดินทางไปทางขวา (+x) y₂ = A·sin(kx + ωt) ← เดินทางไปทางซ้าย (−x) y_รวม = y₁ + y₂ = 2A·cos(ωt)·sin(kx) (คลื่นนิ่ง)

บัพ (Node) — จุดที่ไม่เคลื่อนที่: sin(kx) = 0 → x = 0, λ/2, λ, 3λ/2, ...

ปฏิบัพ (Antinode) — จุดที่สั่นกว้างสุด: |sin(kx)| = 1 → x = λ/4, 3λ/4, ...

🎯 จุดประสงค์การเรียนรู้

1. อธิบายสมบัติของคลื่นผิวน้ำ (A, f, λ, v, T) และความสัมพันธ์ระหว่างกันได้
2. ทดสอบและตรวจสอบสมการ v = fλ จาก simulation ได้
3. อธิบายและระบุแนวปฏิบัพ-บัพในการแทรกสอดของคลื่นได้
4. อธิบายการเกิดคลื่นนิ่งและระบุตำแหน่งบัพ-ปฏิบัพได้

🔬 กิจกรรมที่ 1 — แหล่งกำเนิดเดียว (POE)

P — คาดคะเน: ถ้าเพิ่มความถี่ f โดยคง λ ไว้คงที่ ความเร็วคลื่น v จะเปลี่ยนอย่างไร?
O — สังเกต: เปิด simulation โหมด 🔵 แหล่งเดียว ปรับ f จาก 0.5 → 2.0 Hz โดยคง λ = 80 บันทึกค่า v ที่ได้
E — อธิบาย: ผลที่ได้ตรงกับที่คาดคะเนหรือไม่? สัมพันธ์กับสมการ v = fλ อย่างไร?

🔵🔵 กิจกรรมที่ 2 — การแทรกสอด

2.1 ตั้ง Δφ = 0 (rad), d = 160, λ = 80 สังเกตรูปแบบการแทรกสอด บริเวณใดสว่าง? บริเวณใดมืด? นับจำนวนแนวบัพ (เส้นมืด)
2.2 เปลี่ยน Δφ เป็น π (≈ 3.14 rad) รูปแบบเปลี่ยนแปลงอย่างไร? แนวบัพและปฏิบัพสลับกันหรือไม่?
2.3 คง Δφ = 0, d = 160 แต่เปลี่ยน λ จาก 80 → 40 จำนวนแนวบัพเพิ่มขึ้นหรือลดลง? เพราะเหตุใด?

〰 กิจกรรมที่ 3 — คลื่นนิ่ง

3.1 เปิดโหมด 〰 คลื่นนิ่ง ตั้ง λ = 80 สังเกตตำแหน่งบัพ (N) และปฏิบัพ (AN) ระยะห่างระหว่างบัพสองบัพติดกัน = กี่หน่วย?
3.2 เปลี่ยน λ เป็น 120 หน่วย ระยะห่างระหว่างบัพเปลี่ยนไปอย่างไร? สัมพันธ์กับ λ อย่างไร?
คำนวณ: ถ้า f = 1.5 Hz และ λ = 100 หน่วย จงคำนวณ v, T และระยะห่างระหว่างบัพ
