หลักของฮอยเกนส์ (Huygens' Principle, 1678): ทุก ๆ จุดบนหน้าคลื่น (wavefront) ทำตัวเป็น แหล่งกำเนิดคลื่นย่อยรูปวงกลม (secondary wavelets) แผ่ออกไปด้วยความเร็วเท่ากับคลื่นนั้น · หน้าคลื่นใหม่ในเวลาถัดไปคือ เส้นสัมผัสร่วม (envelope) ของคลื่นย่อยทั้งหมด

① หน้าคลื่นเป็นแหล่งกำเนิดคลื่นย่อย

เลือกจุดหลายจุดบนหน้าคลื่นเดิม WF₀ · ในเวลา Δt แต่ละจุดสร้างคลื่นวงกลมรัศมี r = v · Δt · ลากเส้นสัมผัสด้านหน้าของวงกลมทุกวง → ได้หน้าคลื่นใหม่ WF₁

ซ้าย: หน้าคลื่นระนาบ → wavelet วงกลมหลายวง → envelope = หน้าคลื่นระนาบใหม่ที่ขยับไป r = v·Δt · ขวา: หน้าคลื่นวงกลม → envelope = วงกลมขนาดใหญ่ขึ้น

② หลักการเดียวกันใช้ได้กับคลื่นทุกชนิด

หลักของฮอยเกนส์ไม่ได้จำกัดเฉพาะคลื่นน้ำ — สามารถใช้ได้กับ คลื่นกลและคลื่นแม่เหล็กไฟฟ้าทุกชนิด เพราะอธิบายเฉพาะ การแพร่กระจายของหน้าคลื่น ไม่ใช่ตัวกลาง

💧 คลื่นน้ำ

เห็นได้ชัดในถาดน้ำ (ripple tank): หย่อนแท่งกระทบน้ำเป็นเส้น → เกิดหน้าคลื่นระนาบ · ผ่านช่องแคบ → เลี้ยวเบนเป็นวงกลม

🔊 คลื่นเสียง

เสียงจากปาก/ลำโพงผ่านประตูแคบ → เลี้ยวออกไปกว้าง · เป็นเหตุที่เราได้ยินเสียงจากห้องข้าง ๆ แม้ไม่เห็นแหล่งกำเนิด

💡 คลื่นแสง

แสงผ่านสลิตเล็ก ๆ (≈ λ) → เกิดลายเลี้ยวเบน · ใช้ Huygens อธิบายการหักเหของแสง (กฎ Snell) ได้โดยตรง

📡 คลื่นวิทยุ/ไมโครเวฟ

คลื่นวิทยุเลี้ยวรอบเขา/อาคาร → รับสัญญาณได้แม้ไม่มี line of sight · เป็นการแพร่กระจายตามหลัก Huygens เช่นกัน

ในการสาธิตนี้ ปุ่ม 💧/🔊/💡 เปลี่ยนเพียงสี/สัญลักษณ์ให้เห็นภาพ แต่สมการและการสร้างหน้าคลื่นเหมือนกันทั้งหมด — เพราะธรรมชาติของหน้าคลื่นเป็นไปตามหลักเดียวกัน

③ ทำไม envelope ของ wavelet จึงเป็นหน้าคลื่นใหม่

ถ้าเราพิจารณาเฟสของคลื่นย่อย (wavelet) ทั้งหมด — เฟสที่ตรงกัน (in-phase) จะเสริมกันได้สูงสุดที่ เส้นสัมผัสร่วม ส่วนตำแหน่งอื่น ๆ จะเกิดการหักล้างเฉลี่ยเป็นศูนย์

เมื่อกด 🔗 แสดง envelope ในการสาธิต โปรแกรมจะคำนวณ ขอบนอกของเซ็ตคลื่นย่อยทุกวง โดยตรวจว่าจุดบนวงคลื่นย่อยใดยังคงเป็นจุดนอกสุด (ไม่มีคลื่นย่อยอื่นซ้อนทับ) → ลากเส้นเชื่อม = หน้าคลื่นใหม่ WF₁

  • หน้าคลื่นระนาบ + ไม่มี barrier → envelope ขนานกับ WF₀ เลื่อนไป r = v·Δt → คลื่นเดินตรง
  • หน้าคลื่นวงกลม → envelope = วงกลมรัศมีเพิ่มขึ้น r → ขยายออกจากศูนย์กลางอย่างต่อเนื่อง
  • มี barrier ช่องเปิดกว้าง (d ≫ λ) → envelope ตรงกลางยังเป็นระนาบ ขอบโค้งเล็กน้อย → ใกล้เคียงคลื่นเดินตรง
  • มี barrier ช่องเปิดแคบ (d ≈ λ) → envelope กลายเป็นวงกลมเด่นชัด → เลี้ยวเบน (diffraction)

④ ความเชื่อมโยงไปสู่หัวข้ออื่น

↪ การเลี้ยวเบน (Diffraction)

อธิบายได้ตรง ๆ ด้วย Huygens: เมื่อ d ≈ λ → wavelet ที่ผ่านช่องเปิดมีน้อย → envelope กลายเป็นวงกลมแทนระนาบ

↪ การหักเห (Refraction)

เมื่อหน้าคลื่นเข้าสู่ตัวกลางใหม่ ความเร็วเปลี่ยน → wavelet ในตัวกลางใหม่มีรัศมีต่างกัน → envelope เอียงไปจากเดิม → ได้กฎ Snell: sin θ₁ / sin θ₂ = v₁ / v₂

↪ การสะท้อน (Reflection)

หน้าคลื่นชนผิว → wavelet ใหม่เกิดที่ผิวสะท้อนกลับ → envelope ของ wavelet สะท้อนให้มุมตกกระทบ = มุมสะท้อน

↪ การแทรกสอด (Interference)

คลื่นย่อยจาก 2 source ที่มีเฟสตรงกัน → ตำแหน่งที่ระยะทางต่างกันเป็นจำนวนเต็มของ λ → เสริมกัน (constructive) — เป็นรากฐานของ Young's double slit
