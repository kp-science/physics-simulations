📌 หลักการ: การบวกเวกเตอร์ความเร็ว (Galilean Velocity Addition)

เมื่อวัตถุ (รถ) เคลื่อนที่บนแผ่นที่กำลังเคลื่อนที่ด้วยตัวเอง ความเร็วของรถในกรอบห้องปฏิบัติการ (Lab frame) จะเป็น ผลรวมเวกเตอร์ ของความเร็วสองตัว:

v⃗result = v⃗bulldozer/sheet + v⃗sheet/lab • v⃗result = (vsheet + vbulsinθ) x̂ + (vbulcosθ) ŷ • |v⃗result| = √[ (vsheet + vbulsinθ)² + (vbulcosθ)² ] • α = arctan( vx / vy ) [มุมเบี่ยงจาก +y axis]

นี่คือ สัมพัทธภาพของกาลิเลโอ (Galilean Relativity) ที่ใช้ได้เมื่อความเร็วน้อยกว่าแสงมาก การแปลงกรอบอ้างอิงอย่างง่ายที่สุดในฟิสิกส์คลาสสิก

🔭 Lab Frame (กรอบห้อง)

  • ผู้สังเกตยืนอยู่กับที่บนพื้น
  • เห็นแผ่นเคลื่อนที่ไปทางขวา
  • รถเคลื่อนที่ตามเส้นทแยง
  • วัดได้ v⃗result = ผลรวมเวกเตอร์

🚂 Sheet Frame (กรอบแผ่น)

  • ผู้สังเกตเคลื่อนที่ไปพร้อมกับแผ่น
  • เห็นแผ่นนิ่ง (ห้องเคลื่อนไปซ้าย)
  • รถเคลื่อนที่ตามทิศที่ตั้งไว้ (θ)
  • วัดได้เฉพาะ v⃗bul

🎨 การบวกเวกเตอร์แบบ Head-to-Tail

วางหางของ v⃗bul ที่หัวของ v⃗sheet — เวกเตอร์ผลลัพธ์คือเส้นตรงจากหางแรกถึงหัวสุดท้าย

🔬 สถานการณ์ทดลองทั้ง 3 กรณี

📐 สถานการณ์ที่ 1

รถวิ่งตรงข้ามแผ่น (θ = 0°)
แผ่นเคลื่อน 0.5× ความเร็วรถ
→ รถเบี่ยง α ≈ 26.6° ≈ 27°

📐 สถานการณ์ที่ 2

รถวิ่งตรงข้ามแผ่น (θ = 0°)
แผ่นเคลื่อน 1.0× ความเร็วรถ
→ รถเบี่ยง α = 45.0°

🎯 สถานการณ์ที่ 3

รถหัน ซ้าย 45°
แผ่นเคลื่อน ≈ 0.707× (= 1/√2)
→ รถเคลื่อน ตรงข้ามเลย! α = 0°

🧭 เปรียบเทียบสองกรอบอ้างอิง (vsheet = 0.5×, θ = 0°)

Lab Frame: เห็นรถเคลื่อนทแยง ~27° (เส้นสีเขียว)
Sheet Frame: เห็นรถเคลื่อนตรงข้ามในแนวตั้ง θ = 0° (เส้นสีฟ้า)

💡 สรุปสำคัญ: เส้นทางที่รถ "วาด" บนแผ่นกระดาษ (รอยสีฟ้า) จะเป็นเส้นตรงในทิศที่ตั้ง — แต่เส้นทางในห้อง (รอยสีเขียว) จะเป็นผลรวมเวกเตอร์ ทั้งสองเส้นทางถูกต้องในกรอบของตัวเองเสมอ
