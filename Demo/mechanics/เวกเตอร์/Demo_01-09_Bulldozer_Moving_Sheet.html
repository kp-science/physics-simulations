📖 Demo 01-09: Bulldozer on Moving Sheet (หน้า 26–27)

ใช้ toy bulldozer 2 ตัว ที่มีความเร็วเท่ากัน (v₁) เทียบกับพื้นผิวที่มันอยู่
• Bulldozer A วิ่งบนโต๊ะ ความเร็วในห้องทดลอง = v₁
• Bulldozer B วิ่งบนกระดาษ ซึ่งถูกดึงด้วยความเร็ว v₂ — ความเร็วในห้องทดลอง = v₁ + v₂

นี่คือหลักการ Galilean Velocity Addition — ความเร็วจะบวกกันเมื่อเปลี่ยน frame อ้างอิง

📐 สมการหลัก

v_B(โต๊ะ) = v₁ + v₂ x_A(t) = x_A₀ + v₁ · t x_B(t) = x_B₀ + (v₁ + v₂) · t
กราฟ x-t ของ A: ความชัน = v₁
กราฟ x-t ของ B: ความชัน = v₁ + v₂
เมื่อ v₂ = 0 → ทั้งคู่มีความชันเท่ากัน (ขนานกัน)

🔄 กรณีต่างๆ ของ v₂

▶▶
v₂ > 0 (กระดาษเคลื่อนไปทางเดียวกัน)
v_B = v₁ + v₂ → B วิ่งเร็วกว่า A
▶▶
v₂ = 0 (กระดาษหยุดนิ่ง)
v_B = v₁ → B วิ่งเท่ากับ A (side by side)
◀▶
v₂ < 0 (กระดาษเคลื่อนทิศตรงข้าม)
v_B = v₁ − |v₂| → B วิ่งช้ากว่า A
◀▶
v₂ = −v₁ (กระดาษเร็วเท่ากันแต่สวนทาง)
v_B = 0 → B หยุดนิ่งในห้องทดลอง!

🌍 เปลี่ยน Frame of Reference

v ของ A v ของ B v ของกระดาษ
ห้องทดลอง v₁ v₁ + v₂ v₂
กระดาษ v₁ − v₂ v₁ 0

🛠️ อุปกรณ์ (Equipment)

1. Two battery-powered toy bulldozers (ทำงานที่ความเร็วเท่ากัน)
2. Long sheet of paper/plastic — มีเส้นกริดระยะห่างเท่ากัน
3. Rollers สำหรับดึงกระดาษ (หรือดึงด้วยมือ)

→ กระดาษเปรียบเหมือน frame of reference ที่เคลื่อนที่
→ เส้นกริดบนกระดาษช่วยวัดตำแหน่งใน sheet frame

💡 ตัวอย่างในชีวิตจริง

✈️
เครื่องบิน: ลมหนุน/ต้าน
v_ground = v_plane + v_wind
🏊
ว่ายน้ำในแม่น้ำ:
v_bank = v_swim + v_current
🚶
เดินบนบันไดเลื่อน:
v_floor = v_walk + v_escalator

❓ ข้อ 2: ถ้าดึงกระดาษด้วย v₂ = −4 m/s (ทิศตรงข้าม bulldozer) และ v₁ = 4 m/s ความเร็ว B ในห้องทดลองคือเท่าใด?

❓ ข้อ 3: เมื่อกระดาษหยุดนิ่ง (v₂ = 0) กราฟ x-t ของ bulldozer ทั้งสองจะเป็นอย่างไร?

❓ ข้อ 4: ในมุมมองของกระดาษ (sheet frame) ความเร็วของ bulldozer B คือเท่าใด?
