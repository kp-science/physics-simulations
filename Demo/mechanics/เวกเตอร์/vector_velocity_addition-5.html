การบวกเวกเตอร์ความเร็ว
Vector Velocity Addition · Demo 02-07 · หลักสูตรแกนกลาง 2560 (ฟิสิกส์ ม.4–5)

🔬 หลักการทดลอง

ลูกปิงปองอยู่ในท่อแก้วใสแนวตั้ง เมื่อ ท่อเคลื่อนที่แนวนอน และเชือกที่ยึดลูกบอลกับจุดคงที่ C บนเพดานดึงลูกบอลขึ้น ลูกบอลจะมีความเร็วสองส่วนประกอบพร้อมกัน ผลรวมของทั้งสองคือ เวกเตอร์ความเร็วลัพธ์ ที่ทำมุม θ กับแนวนอน

① ท่อเลื่อนซ้าย → vx

② เชือกดึงขึ้น → vy

③ รวมเวกเตอร์ → |v|

📐 สูตรการคำนวณ

ขนาดความเร็วลัพธ์: |v| = √(vx² + vy²) ทิศทาง (มุม θ): θ = arctan( vy / vx ) ── ตัวอย่าง: vx = vy = 4 m/s ── |v| = √(4² + 4²) = √32 ≈ 5.66 m/s θ = arctan(4/4) = arctan(1) = 45.0° ── ตัวอย่าง: vx = 3, vy = 4 m/s (สามเหลี่ยม 3-4-5) ── |v| = √(3² + 4²) = √25 = 5.00 m/s θ = arctan(4/3) ≈ 53.1°

🔢 เข้าใจทีละขั้น

1
ท่อเลื่อนซ้าย → ลูกบอลได้ vx

เมื่อท่อแก้วเคลื่อนที่ไปทางซ้ายด้วยความเร็ว vx ลูกปิงปองที่อยู่ในท่อจะถูกพาไปพร้อมกัน ลูกบอลจึงมี ความเร็วแนวนอน vx

2
เชือกดึงขึ้น → ลูกบอลได้ vy

เชือกยึดจากลูกบอลไปยังจุดคงที่ Cบนเพดาน เมื่อท่อเลื่อนซ้าย เชือกบังคับให้ลูกบอลเลื่อนขึ้นตามท่อ ลูกบอลจึงมี ความเร็วแนวดิ่ง vy

3
สองส่วนประกอบเกิดพร้อมกัน

vx และ vy เป็นอิสระต่อกัน และเกิดขึ้นพร้อมกัน ความเร็วจริงของลูกบอลคือ เวกเตอร์ลัพธ์ |v| ที่มีทิศทำมุม θ = arctan(vy/vx) กับแนวนอน

4
กรณีพิเศษ vx = vy → θ = 45°

เมื่อส่วนประกอบเท่ากัน สามเหลี่ยมเวกเตอร์จะเป็นสามเหลี่ยมมุมฉากหน้าจั่ว tan(θ) = 1 → θ = 45° เสมอ ไม่ว่า vx=vy จะมีค่าเท่าไรก็ตาม

📊 แผนภาพสามเหลี่ยมเวกเตอร์

ปรับค่า vx และ vy ใน Simulation Tab แล้วกลับมาดูรูปสามเหลี่ยมที่เปลี่ยนไป

💡 ข้อสังเกตสำคัญ

📏 ทฤษฎีบทพีทาโกรัส

|v|² = vx² + vy² สามเหลี่ยมเวกเตอร์มีมุมฉากที่จุดปลาย vx เสมอ เพราะ vx ⊥ vy

🔀 ความเป็นอิสระ

vx และ vy เป็นอิสระต่อกัน หลักการนี้ใช้ได้กับการเคลื่อนที่ 2 มิติทุกประเภท เช่น โพรเจกไทล์

📐 มุมขึ้นกับอัตราส่วน

θ ขึ้นกับ อัตราส่วน vy/vx เท่านั้น (2,2) และ (5,5) ให้ θ = 45° เหมือนกัน
