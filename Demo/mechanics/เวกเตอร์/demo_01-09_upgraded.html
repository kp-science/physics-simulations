💡 สถานการณ์

🔍 กรอบอ้างอิง (Reference Frame)

การวัดความเร็วของวัตถุขึ้นอยู่กับ กรอบอ้างอิง ที่ผู้สังเกตอยู่ ผู้สังเกตคนละกรอบจะวัดความเร็วได้ต่างกัน

ในการสาธิตนี้มีสองกรอบอ้างอิง:

  • กรอบโต๊ะ (Lab Frame) — ผู้สังเกตยืนนิ่งบนพื้น
  • กรอบกระดาษ (Sheet Frame) — ผู้สังเกตอยู่บนกระดาษที่กำลังเคลื่อนที่

📐 หลักการบวกความเร็ว (Galilean Velocity Addition)

เมื่อวัตถุ B เคลื่อนที่ด้วยความเร็ว v₁ เทียบกับกระดาษ และกระดาษเคลื่อนที่ด้วยความเร็ว v₂ เทียบกับโต๊ะ:

v_lab = v₁ + v₂
สถานการณ์v₁ (m/s)v₂ (m/s)v_lab (m/s)

กระดาษเร็วเท่ารถ ทิศตรงข้าม3−30

🎯 สรุปแนวคิดสำคัญ

รถ A เคลื่อนที่บนโต๊ะโดยตรง → ความเร็วใน lab frame = v₁

รถ B เคลื่อนที่บนกระดาษ → ความเร็วในกระดาษ = v₁ แต่กระดาษเองก็กำลังเคลื่อนที่ด้วยความเร็ว v₂ → ความเร็วรวมใน lab = v₁ + v₂

v⃗_BO = v⃗_BS + v⃗_SO

BO = B เทียบกับพื้น, BS = B เทียบกับกระดาษ, SO = กระดาษเทียบกับพื้น
