🔫 Monkey Gun คืออะไร?

การทดลอง Monkey Gun (หรือ Monkey and Hunter) เป็นการสาธิตคลาสสิกที่แสดงให้เห็นว่า องค์ประกอบการเคลื่อนที่แนวนอนและแนวตั้งเป็นอิสระต่อกัน และ แรงโน้มถ่วงกระทำต่อวัตถุทุกชนิดเท่ากัน โดยไม่ขึ้นกับสถานะการเคลื่อนที่เริ่มต้น

🎯 แม้ขวดจะตกลงมาหลังถูกปล่อย กระสุนก็ยังชนขวดเสมอ — ไม่ว่าจะใช้ความเร็วต้น v₀ เท่าใดก็ตาม!

⚙️ กลไกการทดลอง

1. ท่อทองเหลืองเล็งตรงไปยัง ขวดพลาสติก ที่ห้อยอยู่กับ แม่เหล็กไฟฟ้า

2. เมื่อเป่าลมเข้าท่อ ลูกดอกไม้ (กระสุน) ออกไปพร้อมกัน กระแทกแผ่นฟอยล์อลูมิเนียม ตัดวงจรแม่เหล็ก

3. แม่เหล็กดับ → ขวดเริ่มตกอิสระพร้อมกับกระสุนออกจากปาก → ทั้งสองตกภายใต้แรงโน้มถ่วงพร้อมกัน

🟢 ขวด (Falling Object)

ปล่อยจากหยุดนิ่งที่ตำแหน่ง (D+x₀, H_target):

x_bullet(t) = x₀ + v₀·cos θ · t y_bullet(t) = y₀ + v₀·sin θ · t − ½g t² x_bottle(t) = D + x₀ (คงที่) y_bottle(t) = H_target − ½g t²

✅ การพิสูจน์ว่ากระสุนโดนขวดเสมอ

เมื่อกระสุนเดินทางถึงแนวดิ่งใต้ขวด (x = D + x₀) ใช้เวลา:

t_hit = D / (v₀ · cos θ)

เนื่องจากปืนเล็งตรงไปที่ขวด: tan θ = (H_target − H_gun) / D

จึงได้: v₀·sin θ · t_hit = (H_target − H_gun)

y_bullet(t_hit) = H_gun + v₀·sin θ · t_hit − ½g t_hit² = H_gun + (H_target − H_gun) − ½g t_hit² = H_target − ½g t_hit² y_bottle(t_hit) = H_target − ½g t_hit²

∴ y_bullet(t_hit) = y_bottle(t_hit) ✓ กระสุนและขวดอยู่ที่ความสูงเดียวกันเสมอ!

💡 เหตุผลสำคัญ: ทั้งกระสุนและขวดตกลงจากแนวเดิม (ไม่มีแรงโน้มถ่วง) เป็นระยะ ½gt² เท่ากันทุกประการ

⚡ กระสุนตกถึงพื้นก่อนถึง x = D+x₀ หรือขวดตกถึงพื้นก่อนที่กระสุนจะมาถึง → พลาด

เงื่อนไขให้ชน: t_hit < t_ground โดยที่ t_ground = √(2·H_target/g)

v₀_min = D / (cos θ · √(2·H_target/g))

📋 หลักการสำคัญที่ได้เรียนรู้

1. ความเป็นอิสระของการเคลื่อนที่

ความเร็วแนวนอน vₓ = v₀cosθ และแนวตั้ง vᵧ = v₀sinθ − gt ทำงานแยกกัน ไม่กระทบกันและกัน

2. แรงโน้มถ่วงสากล

ทั้งกระสุน (มีความเร็วต้น) และขวด (ตกจากหยุดนิ่ง) ได้รับความเร่ง g = 9.8 m/s² เท่ากันทุกประการ

3. Frame of Reference

ในกรอบที่ "ไม่มีแรงโน้มถ่วง" กระสุนเดินตรงชนขวด — แรงโน้มถ่วงเพียง "เลื่อน" ทั้งสองลงพร้อมกัน

4. SUVAT Connection

การตกของขวด: s = ½gt² ส่วนแนวตั้งกระสุน: Δy = v₀sinθ·t − ½gt² ต่างใช้สมการ s = ut + ½at²
