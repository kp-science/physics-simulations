🎯 จุดประสงค์

ศึกษาความสัมพันธ์ระหว่างองค์ประกอบของแรงโน้มถ่วงบนพื้นเอียงกับมุมเอียง โดยใช้เครื่องชั่งสปริง 2 อัน วัดแรงที่ทำให้รถสมดุล

📐 แผนภาพแรงและสมดุล

สมดุลในแนวขนานพื้นเอียง: F(P) = W · sin θ
สมดุลในแนวตั้งฉากพื้นเอียง: N + F(N) = W · cos θ

∴   N = W · cos θ − F(N)

เมื่อ F(N) = W·cosθ → N = 0   (รถลอยออกจากพื้น)
เมื่อ F(N) = 0       → N = W·cosθ   (ไม่มีสปริงตั้งฉาก)
💡 สังเกตสำคัญ: แรง N (จากพื้นเอียงรองรับรถ) มีทิศตั้งฉากออกจากพื้นเอียงเสมอ เมื่อเราดึงสปริง F(N) มากขึ้น N จะลดลงเท่ากัน เพราะรถยังต้องสมดุลในแนวตั้งฉาก

📈 กราฟ F(P') และ F(N') กับมุมเอียง

W = 12.7 N — เส้นแนวตั้งสีขาว = มุม θ ปัจจุบัน

🟠 F(P')=W·sinθ  |  🟣 F(N')=W·cosθ  |  🔴 W คงที่

F(P') กับ sin θ

สัดส่วนโดยตรง — กราฟ F(P') vs sin θ เป็นเส้นตรงผ่านจุดกำเนิด ความชัน = W

F(N') กับ cos θ

สัดส่วนโดยตรง — กราฟ F(N') vs cos θ เป็นเส้นตรงผ่านจุดกำเนิด ความชัน = W

🧮 ตัวอย่างคำนวณ (Trial 1 จากเอกสาร)

W = 12.7 N   θ = 22.0°   sin22°=0.375   cos22°=0.927

F(P') = 12.7 × 0.375 = 4.76 N   (สปริงวัดได้ 4.5 N → errP ≈ 5.5%)
F(N') = 12.7 × 0.927 = 11.77 N (สปริงวัดได้ 11.5 N → errN ≈ 2.3%)

ถ้า F(N)=0:   N = W·cos22° = 11.77 N  (พื้นรับน้ำหนักทั้งหมด)
ถ้า F(N)=5:   N = 11.77−5.00 = 6.77 N
ถ้า F(N)=11.77: N = 0  (รถลอยออกจากพื้น!)

θ=30° → F(P') = 12.7×0.5 = 6.35 N = W/2 ✓

อุปกรณ์

  • รถไดนามิกส์ (น้ำหนักทราบ)
  • พื้นเอียงปรับมุมได้
  • สปริงสเกล 0–20 N จำนวน 2 อัน
  • เชือก 2 เส้น ≈ 30 cm
  • กระดาษกราฟ

ขั้นตอน

  1. วางรถบนพื้นเอียง ตั้งมุม θ
  2. ผูกเชือก 2 เส้นที่รถ — ต่อสปริงสเกล SP1 (ขนาน↑) และ SP2 (ตั้งฉาก↗)
  3. ดึงทั้งสองพร้อมกันจนรถนิ่ง (สมดุล)
  4. ถอดพื้นเอียงออก — รถยังคงอยู่นิ่ง
  5. อ่านค่า F(P) และ F(N)
  6. เปรียบเทียบกับ Vector Diagram คำนวณ % error
  7. เปลี่ยนมุม θ ทำซ้ำ
