🔵 เวกเตอร์แรงคืออะไร?

แรง (Force) เป็น ปริมาณเวกเตอร์ — มีทั้งขนาด (magnitude) และทิศทาง (direction) จึงไม่สามารถบวกแบบตัวเลขธรรมดาได้ ต้องใช้กฎการบวกเวกเตอร์

รูปที่ 1: 3 แรงดึงที่วงแหวน — เมื่อสมดุล ผลรวมเวกเตอร์ทุกแรง = 0

📐 วิธีการบวกเวกเตอร์

1. วิธีกราฟิก (Graphical Method — Triangle/Parallelogram)

วาดเวกเตอร์ F₁ ก่อน แล้ววาง F₂ ต่อปลาย → เส้นจากต้น F₁ ถึงปลาย F₂ คือ แรงลัพธ์ R

รูปที่ 2: Triangle Method (ซ้าย) และ Parallelogram Method (ขวา)

2. วิธีแยกองค์ประกอบ (Component Method)

แยกองค์ประกอบ: F₁ₓ = |F₁| cos θ₁ F₁ᵧ = |F₁| sin θ₁ F₂ₓ = |F₂| cos θ₂ F₂ᵧ = |F₂| sin θ₂ รวมองค์ประกอบ: Rₓ = F₁ₓ + F₂ₓ Rᵧ = F₁ᵧ + F₂ᵧ ขนาดและทิศทางของแรงลัพธ์: |R| = √(Rₓ² + Rᵧ²) θᵣ = arctan(Rᵧ / Rₓ)

3. Law of Cosines (สูตรตรีโกณมิติ)

|R|² = |F₁|² + |F₂|² + 2|F₁||F₂| cos A โดย A = มุมระหว่าง F₁ และ F₂

⚖️ ภาวะสมดุลสถิต (Static Equilibrium)

วัตถุอยู่ในสมดุลสถิต เมื่อแรงลัพธ์รวมเป็นศูนย์:

ΣF = 0 → ΣFₓ = 0 และ ΣFᵧ = 0 สำหรับ 3 แรง: F₁ + F₂ + F₃ = 0 → F₃ = −(F₁ + F₂) ← F₃ มีขนาดเท่า R แต่ทิศตรงข้าม
รูปที่ 3: เมื่อ 3 แรงสมดุล สามเหลี่ยมแรงจะปิดพอดี และ ΣF = 0

ข้อสังเกตสำคัญ: มุมระหว่าง F₁ และ F₂ ต้องไม่เป็น 180° เพราะถ้า F₁ และ F₂ หักล้างกันหมด แรง F₃ จะเป็นศูนย์ด้วย — แต่ถ้า |F₁| ≠ |F₂| ก็จะไม่มีทางสมดุลได้เลย

🔬 อุปกรณ์และขั้นตอนการทดลอง

ลำดับขั้นตอนจุดสำคัญ
1ตั้งระบบ spring scale 3 อัน บนกระดานแนวนอนตรวจ zero reading ในแนวนอนก่อนเสมอ
2ปรับให้เส้นเชือกยาว 10–20 cm มีค่าต่างกันค่าควรอยู่ตรงกลาง range ของ scale
3ยกวงแหวน แล้วปล่อยให้สมดุลยกขึ้น-วางลง 1-2 ครั้ง เพื่อลด friction
4สอดกระดาษกราฟ จุดตำแหน่งเชือกจุดให้ครบตามแนวเส้นเชือกแต่ละเส้น
5บันทึกค่า F₁, F₂, F₃ (N) และมุม Aหาก scale เป็น gram: ×0.01 เป็น N
6ลากเส้น construct vector F₁, F₂ หาผลลัพธ์ Rใช้ scale เช่น 1 cm = 0.2 N
7เปรียบเทียบ R กับ F₃R ≈ F₃ และทิศตรงข้าม

📊 การคำนวณ % Error

ใช้เปรียบเทียบค่าผลลัพธ์จากกราฟ (R) กับค่าที่วัดได้จาก spring scale ที่ 3 (F₃):

% Error = |R − F₃| / F₃ × 100% ตัวอย่าง: R = 1.47 N, F₃ = 1.39 N % Error = |1.47 − 1.39| / 1.39 × 100% = 5.8%

สาเหตุหลักของ % Error ได้แก่:

  • Spring scale ถูกปรับสอบในแนวตั้ง แต่ใช้ในแนวนอน → ต้อง recalibrate
  • ความไม่แน่นอนในการอ่านค่า (parallax)
  • แรงเสียดทาน (friction) ที่จุดยึด scale
  • ความไม่ตรงของการวาดเส้น (drawing error)
