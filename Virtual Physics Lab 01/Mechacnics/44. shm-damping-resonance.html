📐 ทฤษฎี — Damping & Resonance

1️⃣ Damped SHM — การสั่นที่หน่วง

เมื่อมวล-สปริงมีแรงต้าน (เช่น แรงเสียดทาน, แรงหนืดของของเหลว) สมการการเคลื่อนที่กลายเป็น:

m·d²x/dt² + b·dx/dt + k·x = 0

คำตอบในช่วง under-damped (b² < 4mk):

x(t) = A₀ · exp(−bt/2m) · cos(ω_d · t + φ) ω_d = √(ω₀² − (b/2m)²), ω₀ = √(k/m)

ระบอบของการหน่วง (Damping Regimes)

  • Under-damped (b² < 4mk) — แกว่งหลายรอบก่อนหยุด · ตัวอย่าง: ลูกตุ้มในอากาศ
  • Critically damped (b² = 4mk) — กลับสู่สมดุลเร็วที่สุด ไม่แกว่ง · ตัวอย่าง: โช้คอัพรถยนต์ที่ดี
  • Over-damped (b² > 4mk) — กลับสู่สมดุลช้า ๆ ไม่แกว่ง · ตัวอย่าง: ประตูบานเลื่อนเครื่องสำอาง

2️⃣ Resonance — การสั่นพ้อง

เมื่อมีแรงขับภายนอก F(t) = F₀·cos(ω_drive·t) สมการกลายเป็น:

m·ẍ + b·ẋ + k·x = F₀·cos(ω_drive·t)

แอมพลิจูดในสภาวะคงตัว (steady-state amplitude) — Lorentzian curve:

A(ω) = F₀ / √[(k − mω²)² + (bω)²]

Resonance peak เกิดเมื่อ ω_drive ≈ ω₀ = √(k/m) ทำให้ตัวส่วน (k − mω²) = 0 เหลือเพียง bω → A สูงสุด

3️⃣ Q-factor — คุณภาพของการสั่น

Q = √(km) / b = ω₀·m / b
  • Q สูง (>> 1) — peak แหลม คม ความถี่จำเพาะมาก · เช่น แก้วไวน์ Q ≈ 1000
  • Q ต่ำ (~ 1) — peak กว้าง การสั่นจางเร็ว · เช่น โช้คอัพรถ Q ≈ 0.5
  • FWHM (full-width-half-max) ≈ ω₀ / Q — ใช้วัด Q จากกราฟ resonance

⚠️ Misconception Traps — สิ่งที่นักเรียนมักเข้าใจผิด

M6.1 — "Damping ทำให้ความถี่ลดลง"

ไม่ใช่ทั้งหมด. ในระบอบ under-damped (ที่เห็นในห้องเรียน b เล็ก) ความถี่ damped ω_d = √(ω₀² − (b/2m)²) เปลี่ยน นิดเดียว เมื่อ b เล็ก → สังเกตยาก
สิ่งที่เปลี่ยน ชัดเจน คือ แอมพลิจูด (เกิด exponential decay) ไม่ใช่คาบ

ทดลองยืนยัน: ฐาน A → ตั้ง b = 0, 0.1, 0.5 → จับคาบ T → T แทบไม่ขยับ

M6.2 — "Resonance = การสะท้อน"

ไม่ใช่. Resonance คือ การสะสมพลังงาน เมื่อแรงขับมีความถี่ตรงกับความถี่ธรรมชาติของระบบ ระบบดูดกลืนพลังงานทุกรอบจน A พุ่งขึ้น
การสะท้อน (reflection) คือคลื่นชนสิ่งกีดขวางแล้ววกกลับ — เป็นคนละปรากฏการณ์

ทดลองยืนยัน: ฐาน C → preset Wine Glass → เห็นว่าแก้วสั่นพ้องเพราะ เสียงตรงความถี่ ไม่ใช่เพราะเสียงสะท้อน

M6.3 — "Resonance เกิดเมื่อแรงใหญ่"

ไม่ใช่. ขนาดของแรงขับ F₀ ส่งผลต่อ สเกล ของแอมพลิจูด แต่ ตำแหน่งของ peak (ที่ไหนถึง resonance) ขึ้นกับ ความถี่ เท่านั้น
F₀ ใหญ่แต่ ω_drive ห่าง ω₀ → A ก็ยังเตี้ย

ทดลองยืนยัน: ฐาน B → ตั้ง F₀=5 N, f_drive=0.5 Hz (ห่าง f_natural ≈ 2.25 Hz) → A ≈ น้อยมาก
เทียบกับ F₀=1 N, f_drive=2.25 Hz → A สูงกว่ามาก

4️⃣ การประยุกต์ในชีวิตจริง

🌉 Tacoma Narrows Bridge (1940)

สะพานพังเพราะ aeroelastic flutter — ลมความเร็ว ~67 km/h ทำให้เกิดกระแสน้ำวน (vortex shedding) ที่ความถี่ใกล้ f_natural ของสะพาน (~0.2 Hz) → สะพานสั่นพ้องจน A ใหญ่จนพัง

🏢 Taipei 101 — Tuned Mass Damper

ลูกตุ้มเหล็ก 660 ตันแขวนชั้น 87-92 ปรับ f_natural ของลูกตุ้มให้ตรงกับตึก → เมื่อตึกสั่น ลูกตุ้มสั่นย้อนเฟส → ดูดพลังงานออก ลด A ได้ ~40%

🍷 Wine Glass — Singing Glass

แก้วมี f_natural ขึ้นกับรูปทรงและของเหลว เมื่อเสียงความถี่ตรงกับ f_natural → ผนังแก้วสั่นจน A ทะลุจุดแตก (ใช้ Q สูงของแก้วคริสตัล)

🧲 MRI — Magnetic Resonance Imaging

โปรตอนใน body มี Larmor frequency ขึ้นกับสนามแม่เหล็ก ส่งคลื่น RF ตรงความถี่ → โปรตอน ดูดพลังงาน (resonance) → ปล่อยกลับมาอ่านได้ ใช้ทำภาพ — เป็นการ ใช้ประโยชน์ resonance

🌌 LIGO — Gravitational Wave Detector

กระจกแขวนเป็นลูกตุ้มหลายชั้น (passive isolation) ออกแบบให้ f_natural ต่ำมาก (< 1 Hz) → คลื่นความโน้มถ่วงความถี่สูง (~100 Hz) อยู่ เหนือ resonance → กระจกแทบไม่ถูกรบกวน sensitive ถึง strain ~10⁻²¹

🎡 ชิงช้าเด็ก
