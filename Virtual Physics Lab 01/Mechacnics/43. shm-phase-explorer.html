📐 ทฤษฎี SHM — ครบทุกประเด็น

การเคลื่อนที่ฮาร์มอนิกอย่างง่าย (Simple Harmonic Motion, SHM) คือการเคลื่อนที่กลับไปกลับมาผ่านแนวสมดุล โดย แรงดึงกลับ ∝ การกระจัด และมีทิศตรงข้าม:

F = −ky  ⟹  a = F/m = −(k/m)y = −ω²y

1️⃣ สมการการเคลื่อนที่

การกระจัด
y = A sin(ωt + φ)
A = แอมพลิจูด · φ = เฟสเริ่มต้น
ความเร็ว
v = Aω cos(ωt + φ)
นำหน้า y อยู่ 90°
ความเร่ง
a = −Aω² sin(ωt+φ) = −ω²y
สวนทาง y เสมอ (เฟสต่าง 180°)
y (เขียว) · v (ฟ้า) · a (ชมพู) เฟสต่าง 90°
Phasor หมุน — projection ลงแกน = sin/cos

2️⃣ ตำแหน่งสำคัญในรอบการสั่น

ตำแหน่งy|v||a|คำอธิบายเชิงฟิสิกส์
สมดุล0Aω (สูงสุด)0แรงสุทธิ = 0 ⟹ a = 0 · KE สูงสุด · PE = 0
ปลายสุด +A+A0Aω²หยุดชั่วขณะ · a ทิศ −y · KE = 0 · PE สูงสุด
ปลายสุด −A−A0Aω²เหมือนกัน แต่ a ทิศ +y
y = ±A/2±A/2(√3/2)Aω ≈ 0.866 v_max(½)Aω²KE = ¾ E_total · PE = ¼ E_total
y = ±A/√2±0.707A(1/√2)Aω ≈ 0.707 v_max(1/√2)Aω²KE = PE = ½ E_total (จุดที่พลังงานเท่ากัน)
⚡

สูตรลัดที่ใช้บ่อย — ไม่ต้องรู้ t

v = ω √(A² − y²)  ·  a = −ω²y

พิสูจน์: จาก sin² + cos² = 1 → (y/A)² + (v/Aω)² = 1 → v² = ω²(A²−y²)

3️⃣ คาบและความถี่ของระบบมาตรฐาน

ทั่วไป
ω = 2π/T = 2πf
f = ความถี่ (Hz) · T = คาบ (s)
สปริง-มวล
ω = √(k/m)
T = 2π√(m/k)
ใช้กับสปริงทั้งแนวตั้งและแนวนอน · ไม่ขึ้นกับ A
ลูกตุ้ม (มุมเล็ก)
ω = √(g/L)
T = 2π√(L/g)
ใช้ได้ θ < 15° · ไม่ขึ้นกับ m และ A
ลูกตุ้มมุมเล็ก → SHM (sin θ ≈ θ)
สปริง F = −kx → SHM

4️⃣ ทำไมเฟสต่างกัน 90°?

ถ้า y(t) = A sin θ โดย θ = ωt+φ หมุนสม่ำเสมอ ลองหาอนุพันธ์:

d(sin θ)/dt = ω cos θ  ⟹  v = Aω cos(ωt+φ)
d(cos θ)/dt = −ω sin θ  ⟹  a = −Aω² sin(ωt+φ)

การ diff หนึ่งครั้ง เท่ากับการ เลื่อนเฟส +90° (sin → cos)

  • y → v : เฟส +90° (v นำหน้า y)
  • v → a : เฟส +90° อีก (รวม +180° จาก y)
  • ดังนั้น a = −y × ω² (สวนทางเสมอ)

5️⃣ พลังงานใน SHM

พลังงานจลน์
KE = ½mv² = ½k(A²−y²)
สูงสุดที่สมดุล · = 0 ที่ปลายสุด
พลังงานศักย์
PE = ½ky²
= 0 ที่สมดุล · สูงสุดที่ปลายสุด
รวม (อนุรักษ์)
E = KE + PE = ½kA²
คงที่เมื่อไม่มีแดมป์
⚖️

การแยกพลังงานศักย์ตามวัตถุ (Tab C)

  • ลูกตุ้ม: มีเฉพาะ PE_g (โน้มถ่วง) = mgh ≈ ½mω²y² · ไม่มี PE_s
  • สปริงแนวตั้ง: รวม PE_g (mgy, ±) + PE_s (½ky² − mgy) = ½ky² · พลังงานสะสมใน 2 ที่
  • สปริงแนวนอน: มีเฉพาะ PE_s = ½ky² · แรงโน้มถ่วงตั้งฉาก ⟹ ไม่ทำงาน

6️⃣ การสั่นแบบมีแรงต้าน (Damped SHM)

ถ้ามีแรงต้าน F_drag = −γv (เช่น อากาศ, น้ำ) สมการการเคลื่อนที่กลายเป็น:

m·a + γ·v + k·y = 0  ⟹  ÿ + 2β·ẏ + ω₀²·y = 0  (β = γ/2m)

กรณี under-damped (β < ω₀): สั่นแต่แอมพลิจูดลดลงตาม envelope เลขชี้กำลัง:

y(t) = A·e^(−βt) · sin(ω_d·t + φ)
ω_d = √(ω₀² − β²)  (ความถี่ลู่ลดเล็กน้อย)
Under-damped — แอมพลิจูดลู่เข้าตาม e^(−βt)
พลังงานรวมลด: E(t) = E₀·e^(−2βt)
กรณีเงื่อนไขพฤติกรรม
Under-dampedβ < ω₀สั่นแล้วลู่เข้า (oscillating decay)
Critically dampedβ = ω₀กลับสู่สมดุลเร็วสุดโดยไม่สั่น
Over-dampedβ > ω₀กลับสู่สมดุลช้า ไม่สั่น
⚠️

ความเข้าใจผิดที่พบบ่อย (M4 series)

  • M4.1: "ที่ปลายสุด ความเร่ง = 0" ❌ → จริงคือ สูงสุด (Aω²) เพราะแรงดึงกลับมากสุด
  • M4.2: "ที่สมดุล ความเร่ง = สูงสุด" ❌ → จริงคือ = 0 เพราะแรงสุทธิ = 0
  • M4.3: "v กับ a ทิศเดียวกันเสมอ" ❌ → เฟสต่าง 90° → ทิศเดียวกันเฉพาะบางช่วงของรอบ
  • M4.4: "เคลื่อนไป +y แสดงว่า a ก็ +y" ❌ → a ขึ้นกับ ตำแหน่ง ไม่ใช่ทิศการเคลื่อนที่ (a = −ω²y)
  • M4.5: "คาบของลูกตุ้มขึ้นกับมวล" ❌ → T = 2π√(L/g) ไม่มี m (เพราะ m หักลบกันใน F = ma)
  • M4.6: "แอมพลิจูดมาก คาบนาน" ❌ → คาบ ไม่ขึ้นกับ A ใน SHM (isochronism)

7️⃣ Phase Space (x–v Diagram)

ถ้า plot v บนแกนตั้ง vs y บนแกนนอน → ได้รูปวงรี (สมการ ½ky² + ½mv² = E):

(y/A)² + (v/Aω)² = 1
  • วงรีคงที่ = พลังงานอนุรักษ์ (no damping)
  • หมุน CCW เสมอ (สำหรับ y = A sin)
  • เมื่อมีแดมป์ → spiral เข้าจุดศูนย์กลาง
