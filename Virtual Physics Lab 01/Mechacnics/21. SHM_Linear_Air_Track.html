  4. สังเกตว่าคาบเปลี่ยนแปลงหรือไม่เมื่อเปลี่ยนการกระจัดเริ่มต้น

Part B: ความสัมพันธ์ระหว่างคาบกับมวล

  1. ตั้งค่า K₁ = 2.0 N/m, K₂ = 2.0 N/m, แรงตึง = 0.735 N, การกระจัด = 0.10 m
  2. ตั้งมวล glider = 0.240 kg แล้วทำการทดลอง นับ 20 รอบ
  3. เปลี่ยนมวลเป็น 0.360, 0.480, 0.600 kg ทำซ้ำในแต่ละค่า
  4. สังเกตว่าคาบเปลี่ยนแปลงอย่างไรเมื่อมวลเพิ่มขึ้น

Part C: ความสัมพันธ์ระหว่างคาบกับแรงตึงสปริง

  1. ตั้งค่า K₁ = 2.0 N/m, K₂ = 2.0 N/m, มวล = 0.240 kg, การกระจัด = 0.10 m
  2. ตั้งแรงตึง = 0.735 N แล้วทำการทดลอง นับ 20 รอบ
  3. เปลี่ยนแรงตึงเป็น 0.588, 0.490, 0.343 N ทำซ้ำในแต่ละค่า
  4. สังเกตว่าคาบเปลี่ยนแปลงหรือไม่เมื่อเปลี่ยนแรงตึง

Part D: ความสัมพันธ์ระหว่างคาบกับค่าคงที่สปริง

  1. ตั้งค่า มวล = 0.240 kg, แรงตึง = 0.735 N, การกระจัด = 0.10 m
  2. ตั้ง K₁ = 2.0, K₂ = 1.4 N/m แล้วทำการทดลอง นับ 20 รอบ
  3. เปลี่ยนค่าคงที่สปริงเป็นชุดต่างๆ: (2.0, 2.0), (2.0, 2.5), (2.7, 2.0), (2.5, 2.7)
  4. สังเกตว่าคาบเปลี่ยนแปลงอย่างไรเมื่อค่าคงที่สปริงรวมเพิ่มขึ้น

📊 วิธีการบันทึกผลและวิเคราะห์

  1. บันทึกข้อมูลลงในตารางให้ครบทุก trial

🌊 การเคลื่อนที่แบบฮาร์มอนิกอย่างง่าย (Simple Harmonic Motion) คืออะไร?

เมื่อวัตถุที่ยึดติดกับสปริงถูกดึงออกจากตำแหน่งสมดุลแล้วปล่อย มันจะสั่นไปมารอบตำแหน่งสมดุลด้วยการเคลื่อนที่แบบคาบ เรียกว่า Simple Harmonic Motion (SHM)

คุณสมบัติสำคัญของ SHM:

  • แรงดึงกลับ (Restoring Force) แปรผันตรงกับการกระจัดและมีทิศตรงข้าม: F = -kx
  • การเคลื่อนที่เป็นคาบ — ทำซ้ำทุกๆ คาบ T วินาที
  • คาบการสั่นไม่ขึ้นกับแอมพลิจูด (isochronous)

📐 สมการการเคลื่อนที่

สำหรับระบบสปริง-มวลบนรางลม ที่มีสปริง 2 ตัวยึดทั้งสองข้าง:

x(t) = A · cos(ωt)
v(t) = -Aω · sin(ωt)
a(t) = -ω² · x(t)

โดยที่ ω (ความถี่เชิงมุม) = √((k₁ + k₂) / m)

x, v, a = ตำแหน่ง, ความเร็ว, ความเร่ง ω = ความถี่เชิงมุม (rad/s) A = แอมพลิจูด (m)

⏱ สมการคาบการสั่น

คาบ (Period) คือเวลาที่ใช้ในการสั่นครบ 1 รอบ:

T = 2π √(m / (k₁ + k₂))
T = คาบการสั่น (s) m = มวล glider (kg) k₁ = ค่าคงที่สปริง 1 (N/m) k₂ = ค่าคงที่สปริง 2 (N/m)

🧮 ตัวอย่างการคำนวณ

k₁ = 2.0 N/m, k₂ = 2.0 N/m, m = 0.240 kg

T = 2π√(0.240 / (2.0 + 2.0)) = 2π√(0.240/4.0) = 2π√(0.06) = 2π × 0.2449 = 1.539 s

ค่าทดลอง ≈ 1.5 s (error ≈ 2.5%)

🧮 ตัวอย่างที่ 2

k₁ = 2.0 N/m, k₂ = 2.5 N/m, m = 0.240 kg

T = 2π√(0.240 / (2.0 + 2.5)) = 2π√(0.240/4.5) = 2π√(0.0533) = 2π × 0.2309 = 1.451 s

ค่าทดลอง ≈ 1.4 s (error < 4%)

🔍 ทำไมคาบไม่ขึ้นกับแอมพลิจูด (การกระจัดเริ่มต้น)?

เมื่อแอมพลิจูดเพิ่มขึ้น:

  • ระยะทางที่ต้องเคลื่อนที่มากขึ้น → ใช้เวลามากขึ้น
  • แต่แรงดึงกลับก็เพิ่มขึ้นตามสัดส่วน (F = -kx) → ความเร่งมากขึ้น → เร็วขึ้น
  • ทั้งสองผลหักล้างกันพอดี → คาบคงที่!

สังเกตจากสมการ: T = 2π√(m/(k₁+k₂)) — ไม่มีตัวแปร A (แอมพลิจูด) ในสมการ

🔍 ทำไมคาบไม่ขึ้นกับแรงตึงของสปริง?

แรงตึง (Tension) ของสปริงหมายถึงแรงที่สปริงถูกยืดออกจากความยาวธรรมชาติ ณ ตำแหน่งสมดุล

  • การเปลี่ยนแรงตึงเพียงแค่เลื่อนตำแหน่งสมดุล แต่ไม่เปลี่ยนค่าคงที่สปริง k
  • ค่าคงที่สปริง k คือ ความชัน (slope) ของกราฟ F vs x ซึ่งคงที่ตลอดช่วงการยืดหยุ่น
  • เนื่องจาก k ไม่เปลี่ยน → คาบ T = 2π√(m/k) ก็ไม่เปลี่ยน

💡 เปรียบเทียบ

เหมือนกับการแกว่งลูกตุ้มบนเครื่องบินที่ลอยอยู่ในอากาศ — ไม่ว่าเครื่องบินจะอยู่สูงเท่าไร (เปลี่ยน "ตำแหน่งสมดุล" ของลูกตุ้ม) คาบการแกว่งก็ขึ้นกับความยาวเชือกและ g เท่านั้น

📊 ความสัมพันธ์ระหว่างตัวแปร

T vs m (คาบ vs มวล)

T ∝ √m — คาบเพิ่มตามรากที่สองของมวล

T² vs m (ความสัมพันธ์เชิงเส้น)

T² = 4π²m/(k₁+k₂) — กราฟเส้นตรงผ่านจุดกำเนิด

T vs k_total (คาบ vs ค่าคงที่สปริงรวม)

T ∝ 1/√k — คาบลดเมื่อค่าคงที่สปริงเพิ่ม

T² vs 1/k_total (ความสัมพันธ์เชิงเส้น)

T² = 4π²m · (1/k_total) — เส้นตรงผ่านจุดกำเนิด

⚡ พลังงานใน SHM

พลังงานรวมของระบบ SHM คงที่ตลอดเวลา โดยแลกเปลี่ยนระหว่างพลังงานจลน์ (KE) และพลังงานศักย์ (PE):

E_total = ½kA² = ½mv² + ½kx² = constant
  • ที่จุดปลาย (x = ±A): KE = 0, PE = สูงสุด = ½kA²
  • ที่จุดสมดุล (x = 0): PE = 0, KE = สูงสุด = ½kA²
  • ที่จุดอื่นๆ: KE + PE = ½kA² เสมอ
