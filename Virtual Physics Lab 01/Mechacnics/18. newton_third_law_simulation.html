🎯 จุดประสงค์การทดลอง

สำรวจผลสืบเนื่องของ กฎการเคลื่อนที่ข้อที่ 3 ของนิวตัน โดยการสังเกตวัตถุ 2 ชิ้นที่กระเด็นแยกจากกันในทิศตรงข้าม จากการ "ระเบิด" ที่ควบคุมได้ (แรงสปริงจากลูกสูบที่ถูกกดอัด) และเปรียบเทียบ การเปลี่ยนแปลงของโมเมนตัม ของรถทั้งสองคัน เพื่อยืนยันกฎการอนุรักษ์โมเมนตัม

⚖️ กฎข้อที่ 3 ของนิวตัน (Action–Reaction)

"ทุกแรงกริยา (action) จะมีแรงปฏิกิริยา (reaction) ที่มีขนาดเท่ากันแต่ทิศทางตรงกันข้าม เกิดขึ้นพร้อมกันเสมอ"

ขณะที่ลูกสูบสปริงยืดตัว มันออกแรง F ผลักรถ m₁ ไปทางซ้าย และในเวลาเดียวกันก็ได้รับแรงปฏิกิริยาขนาดเท่ากันจากรถ m₂ ที่ผลักมันไปทางขวา

F₁→₂ = − F₂→₁

แรงทั้งคู่ เกิดขึ้นพร้อมกัน และมี ระยะเวลาที่ออกแรงเท่ากัน (Δt เดียวกัน) ดังนั้น การดล (impulse) ที่กระทำต่อรถทั้งสองคันจะมีขนาดเท่ากัน แต่ทิศทางตรงข้าม

J₁ = F·Δt , J₂ = −F·Δt → |J₁| = |J₂|

🎱 กฎการอนุรักษ์โมเมนตัม

เริ่มต้นรถทั้งสองคันหยุดนิ่ง (โมเมนตัมรวม = 0) และไม่มีแรงภายนอกในแนวระดับ (สมมติให้แรงเสียดทานน้อยมาก) ดังนั้นโมเมนตัมรวมของระบบต้อง คงที่เสมอ:

p_ก่อน = p_หลัง = 0
m₁v₁ + m₂v₂ = 0 ⟹ m₁v₁ = m₂v₂ (ขนาด)

รถที่มี มวลน้อย จะเคลื่อนที่ออกจากจุดศูนย์กลางด้วย ความเร็วสูงกว่า รถที่มีมวลมาก เพื่อให้โมเมนตัมของทั้งสองคันมีขนาดเท่ากัน — เหมือนปืนที่ยิงกระสุน กระสุนพุ่งเร็ว แต่ปืนถอยหลังช้ากว่า

📏 ทำไม m₁·d₁ = m₂·d₂ ?

ในการทดลองนี้เราไม่ได้วัดความเร็ว แต่วัด ระยะทาง ที่รถเคลื่อนที่ได้ก่อนหยุด การที่สมการนี้เป็นจริงมาจากเทคนิคสำคัญ: ผูกเชือกยาว L ระหว่างรถทั้งสองคัน

เชือกยาวทำหน้าที่ หยุดรถทั้งสองคันพร้อมกัน เมื่อผลรวมของระยะทางที่วิ่งได้เท่ากับ L พอดี ดังนั้นเวลาที่รถทั้งสองคันวิ่งจึง เท่ากัน (t₁ = t₂ = t)

เนื่องจากหลังการระเบิด รถเคลื่อนที่ด้วยความเร็วคงที่ (ไม่มีแรงใดทำต่อไป):

d₁ = v₁·t , d₂ = v₂·t

นำกฎอนุรักษ์โมเมนตัม m₁v₁ = m₂v₂ มาคูณทั้งสองข้างด้วย t:

m₁·(v₁·t) = m₂·(v₂·t) ⟹ m₁·d₁ = m₂·d₂
∴ m₁ / m₂ = d₂ / d₁

ข้อดีของวิธีนี้: สามารถหา อัตราส่วนมวล ของรถทั้งสองคันได้โดยไม่ต้องใช้นาฬิกาจับเวลา เพียงวัดระยะทางเท่านั้น — ลดความผิดพลาดจากการจับเวลาลงไปมาก

📉 การวิเคราะห์ % Error

ในทางทฤษฎี อัตราส่วน m₁/m₂ ควรเท่ากับ d₂/d₁ พอดี แต่ในทางปฏิบัติจะมีความแตกต่างเล็กน้อยเนื่องจาก:

  • แรงเสียดทาน ที่ล้อรถ (รถคันหนึ่งอาจเสียดทานมากกว่าอีกคัน → เป็น systematic error)
  • การเคลื่อนที่ไม่เป็นแนวตรง (รถออกนอกแนว → ระยะที่วัดได้ไม่ใช่ระยะจริง)
  • ความผิดพลาดในการวัดระยะ (ความละเอียดของไม้บรรทัด, สายตา → random error)
  • มวลของเชือกยาว เองซึ่งไม่ถูกนับรวม
  • แรงกดของสปริง ที่อาจไม่เสถียรระหว่างการทดลอง
% Error = | (m₁/m₂) − (d₂/d₁) | / (m₁/m₂) × 100 %

จากข้อมูลการทดลองจริงในเอกสารอ้างอิง ค่า % Error ที่ยอมรับได้จะอยู่ในช่วง 3% – 10% (ตัวอย่าง: 2.9%, 4.9%, 5.2%, 8.5%, 9.3%)

หากทุก trial มี error ไปในทิศทางเดียวกัน (เช่น d₂/d₁ ต่ำกว่า m₁/m₂ เสมอ) → แสดงว่ามี systematic error จากอุปกรณ์ ควรตรวจสอบความเรียบของราง และแรงเสียดทานของล้อทั้งสองคัน
