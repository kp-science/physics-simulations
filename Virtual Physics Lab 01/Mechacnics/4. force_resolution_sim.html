📋 วิธีการทดลอง

🎯 จุดประสงค์: นักเรียนเข้าใจการแตกแรงเป็นองค์ประกอบ (Force Resolution) โดยการปรับขนาดแรง F1 และ F2 พร้อมมุมของแต่ละแรง และสังเกตความสัมพันธ์ F1y + F2y = W (สมดุลแนวดิ่ง) และ F1x = F2x (สมดุลแนวนอน)
  1. อ่านค่าเริ่มต้น — สังเกตค่าเริ่มต้น F1 = 4.3 N, A1 = 54°, F2 = 7.3 N, A2 = 79°, W = 10 N บน canvas ด้านซ้ายจะแสดงเวกเตอร์ 3 แรง: F1 (สีม่วง) F2 (สีชมพู) และ W (สีเขียว)
  2. สังเกตองค์ประกอบของแรง F1 — จากข้อมูล 📊 ผลการทดลอง อ่านค่า F1x = F1 cos A1 (แนวนอน สีฟ้า) และ F1y = F1 sin A1 (แนวดิ่ง สีส้ม)
  3. สังเกตองค์ประกอบของแรง F2 — อ่านค่า F2x = F2 cos A2 และ F2y = F2 sin A2 จากตารางข้อมูล
  4. ตรวจสอบสมดุลแนวนอน — จากข้อมูล 🔍 การวิเคราะห์สมดุล ดู ΔFx = |F1x – F2x| และ % error Fx — หากแรงทั้งสองเท่ากันพอดี ค่านี้ควรใกล้ 0%
  5. ตรวจสอบสมดุลแนวดิ่ง — ดู ΣFy = F1y + F2y ว่าเท่ากับ W = 10 N หรือไม่ หากสมดุล ค่า ΔFy = ΣFy – W ควรใกล้ 0 และ % error Fy ต่ำ
  6. ปรับค่า F1 และ A1 — นักเรียนลาก slider F1 หรือ A1 เพื่อเปลี่ยนค่า สังเกตว่า F1x และ F1y เปลี่ยนแปลงอย่างไร และอิทธิพลต่อสมดุลโดยรวม
  7. ค้นหาการสมดุล — ปรับ F2 หรือ A2 จนกว่าตัวบ่งชี้สมดุล (ใน 🔍 การวิเคราะห์สมดุล) แสดงการสมดุลที่ดีในทั้งสองทิศ (% error ต่ำ)
  8. สรุปผล — บันทึกค่าต่างๆ และตอบคำถาม:
    • สูตร Fx = F cos θ และ Fy = F sin θ ใช้ได้ถูกต้องหรือไม่?
    • เมื่อมุม θ เพิ่มขึ้น องค์ประกอบแนวนอน Fx และแนวดิ่ง Fy เปลี่ยนแปลงอย่างไร?
    • เงื่อนไขสมดุล (ΣFx = 0 และ ΣFy = 0) ใช้ได้จริงหรือไม่?
💡 ทดลองเพิ่มเติม: เปลี่ยนค่า W (น้ำหนัก) ไป 5 N หรือ 15 N แล้วปรับ F1 และ F2 ใหม่เพื่อให้ระบบสมดุล สังเกตว่าการเปลี่ยน W ส่งผลต่อการต้องเปลี่ยนแปลง F1, A1, F2, A2 อย่างไร

📐 ทฤษฎี: การแตกแรงเป็นองค์ประกอบ

แรงเป็นปริมาณเวกเตอร์ที่มีทั้งขนาดและทิศทาง แรงเดียวที่กระทำในทิศใดทิศหนึ่ง สามารถแทนด้วยแรงสองแรงที่ตั้งฉากกัน เรียกว่า องค์ประกอบของแรง (Components of Force) ซึ่งประยุกต์ใช้บ่อยที่สุดในการแตกแรงตามแนวแกนนอน (H) และแนวดิ่ง (y)

1. การแตกเวกเตอร์แรงตามแกนตั้งฉาก

เมื่อแรง F ทำมุม θ กับแนวระดับ องค์ประกอบแนวนอน (Fₓ) และแนวดิ่ง (F_y) คือ:

Fₓ = F cos θ (องค์ประกอบแนวนอน / Horizontal Component)
F_y = F sin θ (องค์ประกอบแนวดิ่ง / Vertical Component)

|F| = √(Fₓ² + F_y²) ← ตรวจสอบย้อนกลับ
θ = tan⁻¹(F_y / Fₓ)
F — แรงต้นฉบับ
Fₓ — องค์ประกอบแนวนอน
F_y — องค์ประกอบแนวดิ่ง
องค์ประกอบทั้งสองตั้งฉากกัน และเมื่อรวมกันแบบเวกเตอร์จะได้แรงเดิม: F = Fₓ + F_y

2. เงื่อนไขสมดุล (Equilibrium)

เมื่อระบบอยู่ในสมดุล (ไม่มีการเร่ง) ผลรวมของแรงทุกทิศทางเป็นศูนย์:

ΣFₓ = 0 → F1x = F2x (แนวนอน สมดุล)
ΣF_y = 0 → F1y + F2y = W (แนวดิ่ง สมดุล)

โดย: F1x = F1 cos A1, F2x = F2 cos A2
F1y = F1 sin A1, F2y = F2 sin A2

ในการทดลองจริง ค่าที่วัดได้มักมีความคลาดเคลื่อนเล็กน้อย เนื่องจากความไม่แม่นยำของเครื่องมือวัด สายสปริง และการอ่านมุม คำนวณ % error ได้จาก:

% error (H) = |F1x – F2x| / min(F1x, F2x) × 100%
% error (V) = |ΣFy – W| / W × 100%

3. Free Body Diagram ของจุดสมดุล (Ring)

ที่จุดศูนย์กลาง (ring) มีแรง 3 แรงกระทำ ได้แก่ F1 (ดึงซ้ายบน) F2 (ดึงขวาบน) และ W (ดึงลงตรง) เมื่อระบบสมดุล แรงทั้งสามรวมกันได้เป็นศูนย์

📌 แนวนอน (x)

F1 ดึงไปทางซ้าย, F2 ดึงไปทางขวา → F1x = F2x เพื่อให้สมดุลในแนว x

📌 แนวดิ่ง (y)

F1 และ F2 มีองค์ประกอบพยุงขึ้น ต้านแรงดึงดูดของมวล W → F1y + F2y = W
